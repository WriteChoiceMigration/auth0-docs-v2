Les offres Nuage privé sont des services gérés que vous pouvez utiliser si vous le souhaitez :

* vous ne pouvez pas utiliser un service de nuage public multilocataire dans votre organisation.
* Vous avez des exigences de déploiement géographique pour lesquelles l’option du Nuage public n’est pas réalisable
* ou une capacité supérieure à celle offerte par le nuage public
