Explorez les capacités et les limites des différentes options de nuage privé AWS et Azure.

* Cannot use a multi-tenant Public Cloud service in your organization
* Have geographical deployment requirements where the Public Cloud option isn't feasible
* Require a guaranteed level of requests per second (RPS) or higher capacity than available on Public Cloud
