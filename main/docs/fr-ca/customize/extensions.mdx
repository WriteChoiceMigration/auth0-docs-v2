Les extensions Auth0 vous permettent d’installer des applications ou d’exécuter des commandes/scripts qui étendent les fonctionnalités du produit de base Auth0. Chaque extension est distincte de toutes les autres et est soumise à la [Politique de limite anti-attaques d’Auth0](/support/policies/rate-limit-policy). Auth0 définit des extensions par locataire. Les données sont stockées par paire `locataire\extension`.
