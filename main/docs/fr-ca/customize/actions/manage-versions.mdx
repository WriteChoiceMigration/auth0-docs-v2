Avec l’historique des versions des actions, vous pouvez comparer les versions créées et déployées des actions, et revenir aux versions précédentes si nécessaire.

## Déployer des Actions

Avant qu’une Action puisse être liée à un flux, l’Action doit être déployée.

1. Navigate to [Auth0 Dashboard > Actions > Custom Actions](https://manage.auth0.com/#/actions/library).
2. Choose your Action.
3. Select **Deploy**.

L’Action sélectionnée sera déployée et ajoutée à la collection d’Actions disponibles pour les flux. En outre, une nouvelle version provisoire de l’Action déployée sera créée pour une édition future. Comme les secrets et les dépendances sont liés aux versions, tous les secrets ou dépendances sauvegardés seront disponibles pour la nouvelle version.

## Restaurer des Actions

Si nécessaire, vous pouvez revenir à la dernière bonne version connue d’une Action.

1. Navigate to [Auth0 Dashboard > Actions > Custom Actions](https://manage.auth0.com/#/actions/library).
2. Choose your Action.
3. Select **Version History**.
4. Select the version to which you want to revert, and click **Revert**.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">

   In the version history, “Draft” denotes a version that has never been deployed.

   </Callout>

Une nouvelle copie de la version sélectionnée sera créée et déployée. En outre, une nouvelle version provisoire de l’Action déployée sera créée pour une édition future. Comme les secrets et les dépendances sont liés aux versions, tous les secrets ou dépendances sauvegardés seront disponibles pour la nouvelle version.
