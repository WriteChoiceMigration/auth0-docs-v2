Les intégrations d’authentification unique (SSO) sont des applications client qui permettent d’utiliser des services externes (par ex., [Dropbox](https://marketplace.auth0.com/integrations/dropbox-sso), [Slack](https://marketplace.auth0.com/integrations/slack-sso) ou [Zoom](https://marketplace.auth0.com/integrations/zoom-sso)) pour l’authentification unique. L’intégration permet à vos utilisateurs de se connecter en utilisant les fournisseurs d’identité d’Auth0. Pour voir la liste complète, visitez [Auth0 Marketplace : Intégrations SSO](https://marketplace.auth0.com/features/sso-integrations).

Pour en savoir plus sur les connexions proposées dans Auth0, voir [Connexions Auth0](/connections).
