Vous pouvez configurer des variables globales pour les règles à l’aide de Auth0’s Dashboard. Pour en savoir plus, voir le [Règles d'Auth0](/rules).

1. Allez dans [Auth0 Dashboard > Pipeline Auth > Règles](%24%7Bmanage_url%7D/#/rules) et trouvez la section **Paramètres**.
2. Saisissez une paire clé/valeur de variable et sélectionnez **+****Ajouter**.

La valeur saisie est désormais proposée pour toutes les règles via l’objet de `configuration` global et peut être référencée en utilisant la valeur dans la colonne **Code Snippet**.
