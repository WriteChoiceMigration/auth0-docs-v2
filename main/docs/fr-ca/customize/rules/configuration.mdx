L’objet global `configuration` est disponible dans les règles pour stocker les valeurs couramment utilisées, telles que les URL. Les informations sensibles, telles que les identifiants ou les clés API, doivent être stockées via l’objet `configuration` et ne pas figurer dans le code de vos règles.

## Valeurs de configuration

You can set configuration values in your [Rules Settings](https://manage.auth0.com/#/rules/) on the Dashboard.

Pour modifier ou changer une valeur de clé de configuration, retirez les configurations choisies et remplacez-les par la valeur mise à jour. Vous devez avoir créé au moins une Règle pour voir la section de configuration; sinon, la démo des Règles s'affichera.

## Use the configuration object

Pour modifier ou changer une valeur de clé de configuration, retirez les configurations choisies et remplacez-les par la valeur mise à jour. Vous devez avoir créé au moins une Règle pour voir la section de configuration; sinon, la démo des Règles s’affichera.

```javascript lines
var MY_API_KEY = configuration.MY_API_KEY;
```






The following example is a Rule for sending a Slack message when a new user has signed up. The [Slack Webhook](https://api.slack.com/incoming-webhooks) is a `configuration` value set with the key `SLACK_HOOK_URL`.

```javascript lines
function (user, context, callback) {
  // short-circuit if the user signed up already or is using a Refresh Token
  if (context.stats.loginsCount > 1 || context.protocol === 'oauth2-refresh-token') {
    return callback(null, user, context);
  }

  // get your slack's hook url from: https://slack.com/services/10525858050
  const SLACK_HOOK = configuration.SLACK_HOOK_URL;

  const slack = require('slack-notify')(SLACK_HOOK);
  const message = 'New User: ' + (user.name || user.email) + ' (' + user.email + ')';
  const channel = '#some_channel';

  slack.success({
    text: message,
    channel: channel
  });

  // don’t wait for the Slack API call to finish, return right away (the request will continue on the sandbox)`
  callback(null, user, context);
}
```
