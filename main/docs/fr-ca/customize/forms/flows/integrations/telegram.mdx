## Configurer la connexion à votre coffre-fort

Pour configurer une connexion au coffre-fort pour vos actions Telegram, vous aurez besoin d’un **jeton** :

<Frame>![Dashboard > Actions > Forms > Flows > actions > Telegram](/images/cdy7uua7fh8z/21Q1BjBAgDHxVR0YgAqedA/fcd1c5dfbd5c6f6fe8d9ea4c951efcb3/whatsapp-vault-connection.png)</Frame>

To learn how to generate a token, read [Telegram's Bot API](https://core.telegram.org/#bot-api).

## Envoyer un message Telegram

Envoyer un message à un clavardage Telegram

<Frame>![](/images/cdy7uua7fh8z/kgSIPsWN5djsT1yQqg5Z8/fea0821d0e919c5101d01b84b968dd53/sendtelegrammessage.png)</Frame>

### Paramètres de saisie

<table class="table">
<thead>
<tr>
<th><b>Parameter</b></th>

<tr>
<td>Chat ID (required)</td>

### Paramètres de sortie

You can find the expected response and status codes at [Telegram's documentation](https://core.telegram.org/method/messages.sendMessage).
