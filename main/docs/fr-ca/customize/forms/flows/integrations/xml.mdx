Cette liste d’actions XML vous permet de convertir des objets JSON en chaîne XML et des chaînes XML en objets JSON.

## onvertir JSON en XML

Convertit un objet JSON en chaîne XML.

<Frame>![](/images/cdy7uua7fh8z/3GW2EhBMIX3YDPaqc3rXrZ/7706a863700df51376d645a7d9292a27/convertjsontoxml.png)</Frame>

### Paramètres de saisie

<table class="table">
<thead>
<tr>
<th><b>Parameter</b></th>
<th><b>Description</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>JSON (required)</td>
<td>
              The JSON object that will be converted to an XML string.
          </td>
</tr>
</tbody>
</table>

### Objet de sortie

<table class="table">
<thead>
<tr>
<th><b>Property</b></th>
<th><b>Type</b></th>
<th><b>Description</b></th>
</tr>
</thead>
<tbody>
<tr>
<td><code>xml</code></td>
<td>String</td>
<td>The converted XML string.</td>
</tr>
</tbody>
</table>

### Exemple d’objet de sortie

```json lines
{
  "xml": "<?xml version=\"1.0\" encoding=\"UTF-8\" standalone=\"yes\"?>\n<name>John Doe</name>"
}
```






## Convertir XML en JSON

Convertit une chaîne XML en objet JSON.

<Frame>![](/images/cdy7uua7fh8z/42fO7BelEN4bnlUj70WhV8/e53ada552f7cc05c192466db4777471a/covertxmltojson.png)</Frame>

### Paramètres de saisie

<table class="table">
<thead>
<tr>
<th><b>Parameter</b></th>
<th><b>Description</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>XML (required)</td>
<td>
              The XML string that will be converted to a JSON object.
          </td>
</tr>
</tbody>
</table>

### Objet de sortie

<table class="table">
<thead>
<tr>
<th><b>Property</b></th>
<th><b>Type</b></th>
<th><b>Description</b></th>
</tr>
</thead>
<tbody>
<tr>
<td><code>object</code></td>
<td>Object</td>
<td>The converted JSON object.</td>
</tr>
</tbody>
</table>

### Exemple d’objet de sortie

```json lines
{
  "object": {
    "name": "John Doe"
  }
}
```
