L’extension Débogueur de l’API d’authentification vous permet de tester différents points de terminaison de l’API d’authentification Auth0.

## Installer l’extension

Pour installer cette extension :

1. Accédez à [Auth0 Dashboard > Extensions](%24%7Bmanage_url%7D/#/extensions), puis sélectionnez **Débogueur de l’API d’authentification Auth0**. La fenêtre **Installer l’extension** s’ouvre.
2. Sélectionnez **Installer**.

## Autoriser l’extension

Étant donné que l’extension communiquera avec Management API en votre nom pour récupérer des détails sur les applications que vous avez configurées dans votre Tableau de bord Auth0, vous devrez autoriser son accès.

Pour autoriser l’extension :

1. Si vous suivez ce guide, vous devriez déjà être sur la vue [Extensions installées](%24%7Bmanage_url%7D/#/extensions) à partir de [Auth0 Dashboard > Extensions](%24%7Bmanage_url%7D/#/extensions). Sélectionnez **Débogueur de l’API d’authentification Auth0** pour lancer l’extension. Une boîte de dialogue de consentement s’ouvrira et vous demandera l’accès à votre compte.
2. Consentez à autoriser l’extension à accéder à votre compte. Une fois votre consentement donné, l’extension se lance.

## Configurer l’extension

Une fois l’extension installée, vous devez appliquer la configuration de base. Pour configurer l’extension :

1. Si vous suivez ce guide, vous devriez déjà avoir lancé l’extension. Sélectionnez la vue **Configuration**.
2. Définissez les paramètres de configuration suivants et sélectionnez **Enregistrer** :

* **Domaine** : Le domaine de votre locataire. Ce champ est en lecture seule et s’affiche à titre informatif.
* **Application** : Application pour laquelle vous souhaitez lancer des flux d’authentification. Vous pouvez gérer vos applications à partir de [Auth0 Dashboard > Applications > Applications](%24%7Bmanage_url%7D/#/applications).
* **Callback URL** : L’URL de rappel URL pour cette extension. Vous devez également ajouter cette URL URL dans le champ **URL de rappel autorisées** dans les paramètres de l’application sélectionnée.
* **State** : Informations facultatives relatives à l’état pouvant être envoyées avec le flux d’authentification.
* **Connection** : Nom de la connexion avec laquelle vous souhaitez vous connecter. Vous pouvez utiliser ce paramètre pour contourner l’écran de connexion principal et accéder directement à l’écran de connexion du fournisseur d’identité concerné.

Après avoir configuré l’extension, vous pouvez exécuter n’importe lequel des flux en sélectionnant la vue **OAuth2/OIDC**.
