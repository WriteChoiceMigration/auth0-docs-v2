Utilisez [](/extensions/single-sign-on-dashboard-extension)l’extension Dashboard à authentification unique pour gérer la connexion SSO de vos utilisateurs sur plusieurs applications d’entreprise.

## Prérequis

* [Créez l’application SSO Dashboard](/dashboard/guides/extensions/sso-dashboard-create-app).
* [Installez l’extension SSO Dashboard](/dashboard/guides/extensions/sso-dashboard-install-extension).

## Ajouter les applications

1. Allez à [Auth0 Dashboard > Extensions](%24%7Bmanage_url%7D/#/extensions), et sélectionnez votre nouvelle extension Dashboard SSO. 
Si vous êtes un administrateur, vous pouvez également vous connecter au Dashboard SSO en utilisant `https://${account.tenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
2. Dans le coin droit supérieur, sélectionnez **Paramètres** à partir du menu déroulant situé sous votre nom de locataire.
3. Cliquez sur **Créer application** pour ajouter une nouvelle application.

Le formulaire **Nouvelle application** s’affiche.
4. Remplissez les champs suivant pour la nouvelle application :
5. Sélectionnez **Créer**. Votre nouvelle application s’afficher ensuite sur la page **Applications** du SSO Dashboard avec les autres applications qui auront été créées.
6. Cliquez sur une application pour tester la connexion.
