## Prérequis

Vous devez avoir un compte [Twilio](https://twilio.com/) avec une option de livraison SMS et/ou téléphonique valide.

## Configurer Twilio

1. Connectez-vous à votre compte [Twilio](https://www.twilio.com/login).
2. Accédez à votre [Console Twilio](https://console.twilio.com/). Vous aurez besoin de votre SID Twilio et de votre jeton d’authentification Twilio lors de la configuration de Twilio dans votre Auth0 Dashboard.
3. Accédez à [Auth0 Dashboard > Branding (Image de marque) > Phone Provider (Fournisseur de services téléphoniques)](https://manage.auth0.com/#/branding/phone/provider). Sélectionnez **Twilio** comme votre **fournisseur de services téléphoniques**.
4. Sélectionnez une **Méthode de livraison**. Vous pouvez choisir le **Texte**, la **Voix**, ou les deux comme méthodes de livraison pour vos clients.
5. Pour les **paramètres du fournisseur Twilio**, sélectionnez un **SMS Source**:

1. Si vous sélectionnez **Use From (Utiliser à partir de)**, entrez un numéro de téléphone par défaut à partir duquel les clients recevront des messages téléphoniques.
2. Si vous sélectionnez [Messaging Service (Service de messagerie)](https://www.twilio.com/docs/messaging/services), entrez votre SID du service de messagerie Twilio. Vous pouvez trouver cette valeur dans votre console Twilio.
3. Entrez votre SID Twilio et votre jeton d’authentification Twilio dans la console Twilio. Cliquez sur **Save (Enregistrer)**.

## Téléphone d’essai

Cliquez sur le bouton **Try (Essayer)** pour envoyer un message d’essai par téléphone. Si vous avez configuré avec succès Twilio en tant que fournisseur de services téléphoniques externe, Auth0 enverra un message de confirmation par téléphone en utilisant la méthode de livraison choisie. Si vous ne recevez pas de message par téléphone après quelques minutes, consultez vos [journaux Auth0](https://manage.auth0.com/#/logs) pour voir si des erreurs sont répertoriées.

La page [Messaging Activity (Activité de messagerie)](https://console.twilio.com/us1/develop/sms/overview) de Twilio affiche des informations de livraison pour tous les SMS envoyés à vos utilisateurs.
