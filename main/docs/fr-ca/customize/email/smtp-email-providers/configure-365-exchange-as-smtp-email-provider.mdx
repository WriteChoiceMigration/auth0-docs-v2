Auth0 vous permet de configurer votre propre fournisseur de courriel SMTP, afin que vous puissiez gérer, surveiller et dépanner vos communications par courriel.



## Configurer Microsoft 365

1. Connectez-vous à votre compte [Microsoft Azure](https://azure.microsoft.com/) avec votre compte Microsoft 365.
2. Enregistrez une ressource [Microsoft Entra ID Application (application Microsoft Entra ID)](https://learn.microsoft.com/en-us/azure/healthcare-apis/register-application) si vous n’en avez pas déjà une.
3. Accédez à votre ressource d’application Active Directory. Sélectionnez **Services > App registrations (Enregistrement d’applications)**, puis sélectionnez l’application que vous utiliserez pour envoyer des courriels.
4. Configurez les autorisations d’application requises en accédant à **Manage (Gérer) > API permissions (Autorisation API)**.
5. Ajoutez l’`autorisation`** **[Mail.Send](https://learn.microsoft.com/en-us/azure/healthcare-apis/register-application#api-permissions) en sélectionnant **Add a permission (Ajouter une autorisation) > Microsoft Graph > Application permissions (Autorisations de l’application)**.
6. Récupérer les configurations requises pour le fournisseur :

1. Naviguez jusqu’à **Application Overview (Aperçu de l’application)** pour récupérer [l’ID client et l’ID du locataire](https://learn.microsoft.com/en-us/azure/healthcare-apis/register-application#application-id-client-id) de votre application.
2. Naviguez vers **Manage (Gérer) > Certificates & secrets (Certificats et secrets) **et rassemblez les **valeurs** pour créer un [secret](https://learn.microsoft.com/en-us/azure/healthcare-apis/register-application#certificates--secrets)d’application
3. Récupérez le **courriel de l’utilisateur** de l’organisation qui enverra les courriels par l’intermédiaire de l’application en accédant à **Search (Rechercher) > Users (Utilisateurs)**. Sélectionnez **Services > Users (Utilisateurs)**, sélectionnez votre utilisateur autorisé, puis obtenez son **courriel d’utilisateur** à partir de ses propriétés.
7. Accédez à [Auth0 Dashboard > Image de marque > Fournisseur de messagerie électronique](https://manage.auth0.com/#/branding/email_provider).
8. Activez la bascule **Use my own email provider (Utiliser mon propre fournisseur de messagerie)**.
9. Cliquez sur le logo **Microsoft 365**.
10. Indiquez l’adresse courriel de l’**expéditeur**. Saisissez ensuite l’**ID du locataire**, l’**ID client**, et le **secret client **de l’étape 6, puis cliquez sur **Save (Enregistrer)**.

## Envoyer un courriel de test

Envoyer un courriel de test à l’aide du bouton **Send Test Email (Envoyer un courriel de test)**. Si vous avez tout configuré correctement, vous recevrez un courriel de confirmation.

Si vous ne recevez pas de courriel après quelques minutes, consultez vos [journaux d’Auth0](https://manage.auth0.com/#/logs) pour voir s’il y a des erreurs de code. La page [Email Usage (Utilisation de courriel)](https://admin.microsoft.com/#/reportsUsage/EmailActivity) de Microsoft Exchange affiche désormais des informations sur la livraison de tous les courriels envoyés à vos utilisateurs.
