Auth0 vous permet de configurer votre propre fournisseur de messagerie SMTP afin que vous puissiez gérer, surveiller et dépanner plus en détail vos communications par courriel. L’infrastructure de messagerie intégrée d’Auth0 ne doit être utilisée que pour les courriels de test.

## Prérequis

* Pour que les courriels puissent être envoyés de Auth0 à votre SMTP, des ports doivent être ouverts et il faut autoriser les connexions entrantes provenant d’adresses IP spécifiques. Pour mettre à jour la liste des adresses IP, accédez à [Dashboard > Image de marque](%24%7Bmanage_url%7D/#/branding/email_provider)[>](%24%7Bmanage_url%7D/#/branding/email_provider)[Fournisseur de messagerie électronique](%24%7Bmanage_url%7D/#/branding/email_provider). Voir [Ajouter des adresses IP à AllowList](/security/data-security/allowlist) pour plus de détails.
* Vous devez disposer d’un compte [Mandrill](https://www.mandrill.com/signup/).

## Configurer Mandrill

1. Connectez-vous à [Mandrill](https://www.mandrill.com/signup/).
2. Accédez à [Paramètres](https://mandrillapp.com/settings) dans Mandrill, puis cliquez sur **Ajouter une clé API**. Copiez la valeur de la clé.
3. Accédez à [Auth0 Dashboard > Image de marque> Fournisseur de messagerie électronique](%24%7Bmanage_url%7D/#/branding/email_provider).
4. Activez la bascule **Utiliser mon propre fournisseur de messagerie**.
5. Cliquez sur le logo **Mandrill**.
6. Fournissez une adresse courriel dans le champ **De**, entrez dans le champ **API Key** la clé API Mandrill que vous avez copiée précédemment.
7. Cliquez sur **Enregistrer**.

## Courriel de test

Vous pouvez maintenant envoyer un courriel de test en utilisant le bouton **Envoyer un courriel de test**. Si vous avez tout configuré correctement, vous recevrez un courriel de confirmation. Si vous ne recevez pas de courriel après quelques minutes, veuillez vérifier vos [Journaux Auth0](%24%7Bmanage_url%7D/#/logs) pour toute erreur.

La page [Activités sortantes](https://mandrillapp.com/activity) de Mandrill affichera ensuite des informations de livraison pour tous les courriels envoyés à vos utilisateurs.
