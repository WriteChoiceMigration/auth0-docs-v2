Vous pouvez filtrer vos flux de journaux afin que seuls les événements et catégories d’événements sélectionnés soient transmis. Étapes à suivre pour configurer le filtre d’un flux de journaux :

1. Allez à [Dashboard > Surveillance > Flux](%24%7Bmanage_url%7D/#/log-streams).
2. Sélectionnez un flux de journaux et accédez à l’onglet **Paramètres**.
3. Activez ou désactivez les filtres dans le menu **Filtrer par catégorie d’événement**.
4. Cliquez sur **Enregistrer**
