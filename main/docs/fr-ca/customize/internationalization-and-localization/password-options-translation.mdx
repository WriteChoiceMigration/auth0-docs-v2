Vous pouvez personnaliser la traduction des fonctionnalités de mot de passe Lock suivantes dans votre propre code :

* Fiabilité du mot de passe
* Historique des mots de passe
* Gadget logiciel de modification de mot de passe

Pour personnaliser une de ces fonctionnalités, vous devez inclure la version 1.1 ou ultérieure de la bibliothèque de gadgets logiciels de modification de mot de passe :

`<script src="https://cdn.auth0.com/js/change-password-1.1.min.js"></script>`
