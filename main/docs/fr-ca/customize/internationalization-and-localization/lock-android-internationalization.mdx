Par défaut, **Lock.Android** affiche tout le texte en anglais. Si vous souhaitez afficher le texte dans une autre langue, vous pouvez fournir un fichier `strings.xml` et définir les valeurs à utiliser pour les différents éléments de texte que Lock pourrait afficher.

Android demande à l’appareil quelle langue a été configurée par l’utilisateur et essaie de récupérer la liste des textes traduits pour cette langue. Pour que cela fonctionne, le développeur doit ajouter chaque fichier de traduction dans l’application en utilisant une convention de nommage de dossier spéciale conforme aux normes Android. Des informations supplémentaires peuvent être trouvées dans les [Guides du développeur Android](https://developer.android.com/training/basics/supporting-devices/languages).
