À partir du Dashboard, vous pouvez consulter les informations de journalisation en temps réel pour les hooks (ainsi que pour les règles, les actions, les scripts de base de données personnalisés et les connexions sociales personnalisées).

1. Rendez-vous à [Tableau de bord Auth0 > Pipeline Auth > Hooks](%24%7Bmanage_url%7D/) et sélectionnez l’icône crayon à côté d’un hook.
2. Dans le coin en haut à droite de l’éditeur pour les hooks, sélectionnez l’icône de journaux.
