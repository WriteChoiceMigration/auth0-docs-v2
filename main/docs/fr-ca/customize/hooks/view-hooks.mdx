Pour voir les hooks configurés, vous pouvez les afficher à l’aide du Dashboard ou en récupérer une liste à l’aide de Management API.

## Utiliser Dashboard

Allez à [Tableau de bord Auth0 > Pipeline Auth > Hooks](%24%7Bmanage_url%7D/).

Tous les hooks configurés sont répertoriés en fonction du point d’extensibilité auquel ils sont exécutés. Un point vert à côté d’un hook indique qu’il est activé.



## Utiliser Management API

Effectuez un appel `GET` au point de terminaison [Obtenir les hooks](/api/management/v2/#!/Hooks/get_hooks). Veillez à remplacer le paramètre fictif `MGMT_API_ACCESS_TOKEN` par votre jeton d’accès à Management API.
