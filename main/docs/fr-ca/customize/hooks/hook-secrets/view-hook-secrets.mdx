Vous pouvez voir les secrets de hook configurés pour un hook à l’aide de Dashboard ou récupérer une liste de secrets de hook à l’aide de Management API.

## Afficher les secrets de hook à l’aide de Dashboard

1. Allez à [Tableau de bord Auth0 > Pipeline d'auth > Crochets](%24%7Bmanage_url%7D/) et cliquez sur l’icône de crayon en regard du hook dont vous voulez afficher les secrets.
2. Dans l’Éditeur de hook, cliquez sur l’icône de clé à molette, puis cliquez sur **Secrets**.

Tous les secrets configurés pour le hook sélectionné sont indiqués.

## Obtenir les secrets de hook en utilisant Management API

Lancez un appel `GET` au [point de terminaison Obtenir les secrets d’appel](/api/management/v2/#!/Hooks/get_secrets). Assurez-vous de remplacer les valeurs des paramètres fictifs `HOOK_ID` et `MGMT_API_ACCESS_TOKEN` avec l’ID de votre appel et votre jeton d’accès de Management API.
