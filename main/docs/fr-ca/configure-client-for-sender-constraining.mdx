title: Configurer le client pour la contrainte de l’expéditeur
---



Lorsque vous exigez une contrainte de l’expéditeur pour une application client, les jetons d’accès sont limités à cette application. Auth0 vérifie les demandes pour s’assurer que seule l’application ayant demandé le jeton peut l’utiliser pour accéder à la ressource associée. Pour en savoir plus, veuillez consulter [Configurer la contrainte d’expéditeur](/configure-sender-constraining).
