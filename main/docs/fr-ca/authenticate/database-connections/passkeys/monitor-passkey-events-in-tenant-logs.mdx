Vous pouvez surveiller les flux de clés de chiffrement associés à vos connexions de base de données par l’intermédiaire des [journaux des locataires](/logs).

Vous pouvez afficher les événements de journaux dans Auth0 Dashboard ([Surveillance> Journaux](https://manage.auth0.com/#/logs)) ou récupérer des journaux en utilisant [Management API](https://auth0.com/docs/api/management/v2/logs/get-logs).
