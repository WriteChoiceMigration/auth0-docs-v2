## Activer la configuration des attributs

1. Naviguez vers [**Auth0 Dashboard > Authentication > Base de données**](%24%7Bmanage_url%7D/#/connections/database), puis sélectionnez la Connexion de base de données que vous souhaitez activer dans la liste de **Connexions de base de données**.
2. Allez sur l’onglet **Attributs** et choisissez **Activer** sur l’écran **Nouvelle configuration des attributs**, puis confirmez que vous avez testé votre configuration dans un environnement de développement.
3. Sélectionnez **Procéder**. Les nouvelles options de configuration sont disponibles.

### Choisissez attributs sur une nouvelle connexion

Les attributs sont ajoutés aux connexions en naviguant vers [**Auth0 Dashboard > Authentification > Base de données**](${manage_url}/#/connections/database).

Pour créer une nouvelle connexion, choisissez **+ Créer une connexion à la base de données** et sélectionnez les Attributs à ajouter. Pour ajouter, configurer ou supprimer des Attributs utilisés par une connexion existante, configurez une connexion existante, choisissez l’onglet **Attributs** et suivez les instructions ci-dessous.

Les valeurs par défaut sont attribuées à une nouvelle connexion et peuvent être modifiées après la création de la connexion.



### Ajouter des Attributs à une connexion existante

Naviguez jusqu’à l’onglet **Attributs** et choisissez **Activer**, puis choisissez parmi les attributs disponibles et sélectionnez l’option **Créer**.



### Configurer ou supprimer des Attributs existants sur une connexion

Tous les Attributs peuvent être modifiés ou supprimés à partir de votre Auth0 Dashboard. Les utilisateurs ne peuvent pas se connecter, s’inscrire ou réinitialiser leur mot de passe en utilisant l’Attribut supprimé, et un minimum d’un Attribut doit rester actif sur chaque connexion.

Pour modifier ou supprimer un Attribut, naviguez jusqu’à l’onglet Attributs, puis cliquez sur le bouton de menu **…** à côté d’un Attribut et sélectionnez **Configurer** ou **Supprimer** dans le menu déroulant.

### Rétablir une connexion

Pour rétablir l’état initial d’une connexion, utilisez le point de terminaison [Update a connection](https://auth0.com/docs/api/management/v2/connections/patch-connections-by-id) de Management API avec un objet `options` qui n’inclut pas le paramètre `attributes`, comme indiqué ci-dessous.
