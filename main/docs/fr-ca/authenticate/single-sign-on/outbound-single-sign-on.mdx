Dans le cas de l’authentification unique (SSO) initiée par un fournisseur d’identité, un fournisseur d’identité tiers est le fournisseur de l’authentification unique. Lorsqu’un utilisateur se connecte à une application :

1. L’application redirige l’utilisateur vers un fournisseur d’identité.
2. Le fournisseur d’identité tiers effectue l’authentification et l’autorisation
3. Lorsqu’une connexion est réussie, l’utilisateur est renvoyé vers l’application.
