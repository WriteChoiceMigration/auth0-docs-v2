Utilisez la configuration SAML suivante pour Hosted Graphite.

1. Accédez à [Dashboard > Applications > Applications](%24%7Bmanage_url%7D/#/applications) et créez une nouvelle application ou cliquez sur le nom d’une application à mettre à jour.
2. Accédez à l’onglet **Modules complémentaires** et activez **l’Application Web SAML2** .
3. Dans l’onglet **Paramètres**, définissez l’**URL de rappel de l’application** sur : `https://www.hostedgraphite.com/complete/saml/{yourUserId}/`.
4. Collez le code suivant dans la zone de texte **Paramètres** et cliquez sur **Déboguer**.
5. Faites défiler vers le bas de la page puis cliquez sur **Activer**.
6. Dans l’onglet **Usage**, localisez **Métadonnées du fournisseur d’identité**, et cliquez sur **Télécharger** pour télécharger le fichier de métadonnées. Vous en aurez besoin lorsque vous configurerez Auth0 comme fournisseur d’identité.
