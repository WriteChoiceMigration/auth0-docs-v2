Le protocole Security Assertion Markup Language (SAML) est un cadre d’applications basé sur XML et à norme ouverte pour l’authentification et l’autorisation entre deux entités sans mot de passe.

* Le**fournisseur de services** (SP) accepte de faire confiance au fournisseur d’identité pour authentifier les utilisateurs.
* Le**fournisseur d’identité** (IdP) authentifie les utilisateurs et fournit aux fournisseurs de services une assertion d’authentification indiquant qu’un utilisateur a été authentifié.

Auth0 prend en charge le protocole SAML et peut servir de fournisseur d’identité (IdP), de fournisseur de services (SP) ou des deux, pour ce qui suit :

* Applications Web SAML2
* Intégrations SAML SSO
* SSO initié par l’IdP
* Services de fédération de Microsoft Active Directory (ADFS)
* Signature et chiffrement des demandes SAML

## Liaisons et options SAML prises en charge

Auth0 prend en charge les liaisons SAML suivantes :

* HTTP Redirect
* HTTP POST

Auth0 prend en charge les options SAML suivantes :

* Profil SSO du navigateur Web
* Profil de déconnexion unique
* Profil de gestion de l’identifiant de nom
* Profil de mappage de l’identifiant de nom
