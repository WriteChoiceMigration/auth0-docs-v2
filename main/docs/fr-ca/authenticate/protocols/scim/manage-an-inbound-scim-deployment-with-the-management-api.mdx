Pour en savoir plus sur les opérations de gestion SCIM prises en charge et comment en faire usage, allez à [Explorateur de Management API Auth0](https://auth0.com/docs/api/management/v2) et développez l’onglet **Connections (Connexions)**. Management API peut activer, mettre à jour et supprimer la configuration SCIM pour une connexion sélectionnée, ainsi que créer et supprimer des jetons SCIM.

### **Enregistrez une application client pour la communication de Management API**

Pour utiliser Management API Auth0, enregistrez une application client de communication entre machines disposant des autorisations nécessaires pour gérer votre configuration SCIM.

1. À partir d'[Auth0 Dashboard](%24%7Bmanage_url%7D/#/), sélectionnez **Applications**.
2. Sélectionnez **Créer une application**.
3. Saisissez un nom, sélectionnez **Application Communication entre machines**, et sélectionnez **Créer**.
4. Sur votre nouvelle application, sélectionnez l’onglet **API**, et réglez **Management API d’Auth0** sur **Autorisé**.
5. Sous **Permissions**, sélectionnez `read:scim_config`, `create:scim_config`, `update:scim_config`, `delete:scim_config`, `create:scim_token`, `read:scim_token`, et `delete:scim_token.`
6. Sélectionnez **Mise à jour**.
7. Suivez les instructions dans [Jetons d’accès à Management API](/security/tokens/access-tokens/management-api-access-tokens) pour apprendre comment obtenir les jetons et appeler les points de terminaison de [Management APIv2 Auth0](https://auth0.com/docs/api/management/v2/connections/post-scim-configuration).
