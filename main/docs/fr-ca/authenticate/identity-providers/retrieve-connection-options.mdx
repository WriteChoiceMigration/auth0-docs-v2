Vous pouvez récupérer l’objet d’options pour une [connexion](/connections) à l’aide de Management API Auth0.

Effectuez un appel `GET` au point de terminaison [Obtenir une connexion](/api/management/v2#!/Connections/get_connections_by_id). Veillez à remplacer les valeurs d’espace réservé `CONNECTION_ID` et `MGMT_API_ACCESS_TOKEN` par votre ID de connexion et votre jeton d’accès à Management API, respectivement.
