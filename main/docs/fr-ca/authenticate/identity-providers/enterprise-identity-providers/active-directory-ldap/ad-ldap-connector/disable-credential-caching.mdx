Vous pouvez désactiver la mise en cache des identifiants au niveau de la connexion des [connexions AD/LDAP](/extensions/ad-ldap-connector) à l’aide d'Auth0 Dashboard.

1. Allez dans [Auth0 Dashboard > Authentification > Enterprise](%24%7Bmanage_url%7D/#/connections/enterprise), et sélectionnez le type de connexion **Active Directory/LDAP.**
2. Sélectionnez le nom de la connexion que vous souhaitez modifier.
3. Activez le bouton **Désactiver le cache**.
