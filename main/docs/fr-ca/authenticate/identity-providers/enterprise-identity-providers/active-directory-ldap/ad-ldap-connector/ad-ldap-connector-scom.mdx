Le connecteur Auth0 AD/LDAP peut s’exécuter en tant que service sur des machines Windows.

Vous pouvez surveiller l’état du service à l’aide de System Center comme vous le feriez avec n’importe quel autre service.

1. Open the **Add Monitoring Wizard** and select the **Monitoring Type: Windows Service**:

   <Frame>![Monitor AD/LDAP Connector Wizard Screen 1](/images/cdy7uua7fh8z/6234mP3Hx1g3Iwf7y9mKH9/7a2ac217f7d22070c20a769939461f75/ss-2014-12-11T22-48-51.png)</Frame>
2. Enter a name and description:

   <Frame>![Monitor AD/LDAP Connector Wizard Screen 2](/images/cdy7uua7fh8z/2KZkQoBuubAgAEWBPnkJx0/abdcdf5e1e0fad20829b255401db523f/ss-2014-12-11T22-49-57.png)</Frame>
3. Select the Server in which the AD/LDAP Connector is installed and then choose **Auth0 ADLDAP**:

   <Frame>![Monitor AD/LDAP Connector Wizard Screen 3](/images/cdy7uua7fh8z/DOQHiJYYiJKZrWzHYUIiB/21eae1ad8435a0754c6e221d5e33bab5/ss-2014-12-11T22-50-37.png)</Frame>
4. Select the limits of **CPU** and **Memory limits**. For example, 10% of CPU and 200MB of RAM are good limits to trigger alerts.

Nous vous recommandons également de configurer une transaction synthétique pour surveiller l’authentification de bout en bout.

To learn more, see [Monitor Auth0 Using SCOM](/docs/deploy-monitor/monitor/monitor-using-scom).
