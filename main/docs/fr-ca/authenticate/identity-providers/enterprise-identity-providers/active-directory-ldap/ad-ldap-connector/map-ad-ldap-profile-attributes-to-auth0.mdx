Vous pouvez modifier la correspondance des attributs de profil entre les attributs AD/LDAP et les attributs du profil utilisateur Auth0.

1. À partir d'un navigateur, rendez-vous sur [http://localhost:8357](http://localhost:8357) pour lancer la **console d'administration** du connecteur sur le serveur du connecteur AD/LDAP.
2. Cliquez sur **Mappeur de profil** pour ouvrir un écran de l’éditeur de code qui fait correspondre les attributs entre un service de répertoire source (représenté par la variable `raw_data`) et une variable renvoyée pour remplir le profil utilisateur Auth0. 
La première partie de la fonction crée une instance de la variable appelée `profile` et possède une correspondance pour la partie principale du profil utilisateur Auth0.  Des attributs supplémentaires peuvent être définis en dessous en utilisant la syntaxe suivante dans le formulaire :
`profile['department'] = raw_data['companydept'];`
Dans cet exemple, `department` est le nom de l’attribut dans le profil utilisateur Auth0 et `companydept` est le nom de l’attribut dans le service du répertoire source (comme AD).
3. Cliquez sur **Enregistrer** lorsque vous avez terminé de modifier la configuration LDAP.
4. Une fois que le connecteur a exécuté les tests, assurez-vous que tous les tests sont en vert.
