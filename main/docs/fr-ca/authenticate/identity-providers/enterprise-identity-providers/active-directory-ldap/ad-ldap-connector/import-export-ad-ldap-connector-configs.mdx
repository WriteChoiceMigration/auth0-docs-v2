Vous pouvez exporter la configuration du connecteur AD/LDAP ou importer une configuration précédemment exportée. Cette fonction est utile pour les déploiements comportant plusieurs nœuds du connecteur AD/LDAP déployés à des fins de haute disponibilité. Vous pouvez configurer et tester la configuration sur un nœud, puis exporter la configuration de travail et l’importer dans tous les nœuds suivants.

1. À partir d'un navigateur, rendez-vous à [http://localhost:8357](http://localhost:8357) pour lancer la **console d'administration du connecteur** sur le serveur du connecteur AD/LDAP.
2. Cliquez sur l'onglet **Importation/Exportation**.
3. Exportez un fichier .zip contenant le fichier `config.json`, le dossier `certs` et le fichier `lib\profileMapper.js` du dossier `%Program Files(x86)%\Auth0\AD LDAP Connector\`.
4. Pour importer, téléchargez un fichier .zip de sauvegarde dans l'onglet **Importation/Exportation**.
