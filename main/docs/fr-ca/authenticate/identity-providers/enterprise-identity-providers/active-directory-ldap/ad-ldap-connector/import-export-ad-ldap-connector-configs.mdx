Vous pouvez exporter la configuration du connecteur AD/LDAP ou importer une configuration précédemment exportée. Cette fonction est utile pour les déploiements comportant plusieurs nœuds du connecteur AD/LDAP déployés à des fins de haute disponibilité. Vous pouvez configurer et tester la configuration sur un nœud, puis exporter la configuration de travail et l’importer dans tous les nœuds suivants.

1. From a browser, go to `http://localhost:8357` to launch the **Connector Admin Console** on the AD/LDAP Connector server.
2. Click **Import / Export** tab.
3. Export a .zip file that contains the `config.json` file, the `certs` folder and the `lib\\profileMapper.js` file from the `%Program Files(x86)%\Auth0\AD LDAP Connector\` folder.
4. To import, upload a backup .zip file on the **Import / Export** tab.
