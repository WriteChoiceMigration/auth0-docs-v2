Vous pouvez activer les connexions d’entreprise pour les applications à l’aide d'Auth0 Dashboard. Pour activer votre connexion d’entreprise, vous devez déjà l’avoir configurée.

1. Allez dans [Auth0 Dashboard > Authentification > Entreprise](%24%7Bmanage_url%7D/#/connections/enterprise) et sélectionnez le type de connexion à afficher.
2. Sélectionnez le nom de la connexion à afficher.
3. Sélectionnez l’affichage **Applications**, et activez ou désactivez la connexion pour la ou les applications appropriées.

Lorsque vous configurez votre connexion IdP d’entreprise, rappelez-vous que le paramètre de `connection` passé au point de terminaison `/authorize, ``/whr, `ou` /wsfed` Auth0 se redirigera vers l’IdP amont, **sauf** si la connexion est effectuée via une base de données ou Active Directory.
