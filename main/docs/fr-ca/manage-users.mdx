La base de données hébergée dans le nuage d’Auth0 stocke une variété d’informations sur vos utilisateurs qui vous sont accessibles. Ces informations sont accessibles sur le profil utilisateur, et vos utilisateurs sont regroupés par locataire. Les informations relatives aux utilisateurs peuvent provenir de différentes sources, dont les fournisseurs d’identité, vos propres bases de données et les connexions d’entreprise (Active Directory, SAML).
