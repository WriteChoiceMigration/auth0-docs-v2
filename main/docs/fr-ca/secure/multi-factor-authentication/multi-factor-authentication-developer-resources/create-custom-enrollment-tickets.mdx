Vous pouvez gérer les inscriptions des utilisateurs en créant des billets d’inscription à l’aide du point de terminaison [`/Guardian/post_ticket`](https://auth0.com/docs/api/management/v2#!/Guardian/post_ticket) de l’API MFA. Cette API renverra un billet d’inscription contenant un `ticket_id` et une `ticket_url`, qui peuvent être utilisés pour inscrire un utilisateur. La `ticket_url` ne peut être utilisée qu’une seule fois. Les inscriptions suivantes doivent être effectuées par l’intermédiaire de l’API MFA.

Si vous utilisez la connexion universelle classique et que vous avez besoin de personnaliser l’apparence de la page lorsque l’utilisateur navigue vers la `ticket_url`,vous pouvez éditer la page MFA.
