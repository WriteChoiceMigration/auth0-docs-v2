Utilisez l’authentification multifacteur (MFA) adaptative pour activer la MFA lorsque Auth0 détermine qu’une tentative de connexion est à risque et pour enregistrer les évaluations de risque pour toutes les transactions de connexions de vos bibliothèques de locataires.

## Activer l’authentification multifacteur (MFA) adaptative

Vous pouvez activer l’authentification multifacteur (MFA) adaptative dans le Auth0 Dashaboard ou avec Management API.



## Activer l’évaluation de risque Authentification multifacteur (MFA) adaptative

Si vous n’êtes pas prêt à activer l’authentification multifacteur (MFA) adaptative mais que vous souhaitez commencer à l’entraîner pour analyser les comportements de connexion, vous pouvez activer l’évaluation de risque de l’authentification multifacteur (MFA) adaptative indépendamment.

1. Allez dans [Dashboard > Sécurité > Multi-factor Auth](%24%7Bmanage_url%7D/#/security/mfa).
2. Localisez la section **Définir les politiques**.
3. Dans **Évaluateurs du risque pour la MFA**, sélectionnez **Activer l’évaluation de risque de l’authentification multifacteur (MFA) adaptative**.
4. Sélectionnez **Enregistrer**.

## Personnaliser l’authentification multifacteur (MFA) adaptative

Vous pouvez personnaliser le comportement de l’authentification multifacteur (MFA) adaptative pour fournir la meilleure expérience pour vos utilisateurs tout en assurant leur sécurité. Pour en savoir plus, lisez [Personnaliser l’authentification multifacteur (MFA) adaptative](/mfa/adaptive-mfa/customize-adaptive-mfa).



## Limites

Les informations d’évaluation dans les journaux de locataires ne sont disponibles que pour les flux interactifs Auth0 ne prend pas en charge l’enregistrement des informations d’évaluation pour les flux d’autorisation de mot de passe de propriétaire de ressources (Resource Owner Password Grant, ROPG) sans l’authentification multifacteur (MFA) adaptative. Pour plus d’informations sur les limitations du flux d’authentification, lisez [Authentification multifacteur (MFA) adaptative](/mfa/adaptive-mfa).
