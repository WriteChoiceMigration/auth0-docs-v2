Lorsque vous utilisez votre propre code HTML pour les pages MFA avec la connexion classique, le gadget logiciel MFA d’Auth0 présente les limitations suivantes :

* La MFA par courriel n’est pas prise en charge.
* Si les utilisateurs ont enregistré plusieurs facteurs, ils ne peuvent pas sélectionner celui à utiliser; le gadget logiciel MFA les invite à se connecter avec le facteur le plus sécurisé.
* Les fonctionnalités d’[internationalisation](/brand-and-customize/internationalization-and-localization/universal-login-internationalization) de la connexion universelle ne sont pas utilisées.

Les options de thème pour le gadget logiciel MFA sont répertoriées sous la propriété `theme`.
