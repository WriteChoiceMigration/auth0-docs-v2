Si vous utilisez des flux de connexion et d’inscription sans mot de passe, vous pouvez activer la détection de robots pour ajouter une étape CAPTCHA dans les scénarios où une demande est déterminée comme étant à haut risque par Auth0.



## Configurer la détection de robots pour les flux sans mot de passe

Vous pouvez utiliser l’Auth0 Dashboard pour configurer le comportement de la fonction CAPTCHA.

1. Dans l’[Auth0 Dashboard](https://manage.auth0.com/dashboard), accédez à [Sécurité > Protection contre les attaques > Détection de robots](https://manage.auth0.com/dashboard/#/security/attack-protection/bot-detection).
2. Naviguez vers la section **Réponse**.
3. Pour le paramètre **Implémenter CAPTCHA pour les flux sans mot de passe**, sélectionnez quand requérir CAPTCHA :

* **Jamais** : Les utilisateurs ne sont pas tenus de remplir un CAPTCHA pour se connecter.
* **Lorsque c’est risqué** : Les utilisateurs doivent remplir un CAPTCHA si la connexion présente un risque élevé.
* **Toujours** : Les utilisateurs doivent toujours remplir un CAPTCHA pour se connecter.

## Configurer la détection de robots pour les pages de connexion personnalisées

Si vous utilisez des pages de connexion personnalisées et que vous souhaitez configurer la détection de robots, veuillez lire [Ajouter la détection des robots aux pages de connexion personnalisées](/attack-protection/bot-detection/bot-detection-custom-login-pages).
