Pour intégrer Auth0 à une application native, vous devez d’abord enregistrer votre application auprès d’Auth0 à l’aide de Auth0 Dashboard. Il s’agit d’applications mobiles, de bureau ou hybrides fonctionnant de manière native sur un appareil (par exemple, iOS, Android).

1. Accédez à [Auth0 Dashboard > Applications > Applications](%24%7Bmanage_url%7D/#/applications), puis sélectionnez **Créer une application**.
2. Saisissez un nom descriptif pour votre application, sélectionnez **Native**, et cliquez sur **Créer**.

Une fois la nouvelle application créée, vous serez dirigé vers **Application****Paramètres**, qui comprend les onglets suivants : 
Pour en savoir plus, consultez [Paramètres de l’application](/applications/application-settings).
3. Si vous développez une application mobile, fournissez les paramètres iOS/Android nécessaires dans la zone **Paramètres avancés**, puis cliquez sur **Enregistrer les modifications**.

* Pour les applications iOS, [fournissez votre **ID d’équipe** et **Identifiant du groupe d’applications**](/applications/enable-universal-links-support-in-apple-xcode).
* Pour les applications Android, [fournissez votre **Nom du paquet d’applications** et votre **Hachages de clés**](/applications/enable-android-app-links-support).
