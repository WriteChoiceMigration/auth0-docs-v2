Pour intégrer Auth0 à une application Web classique, vous devez d’abord enregistrer votre application auprès d’Auth0 à l’aide de Auth0 Dashboard. Ces applications peuvent inclure des applications Web classiques qui exécutent la leur logique d’application principalement sur le serveur (comme, Express.js ou ASP.NET).

1. Accédez à [Auth0 Dashboard > Applications > Applications](%24%7Bmanage_url%7D/#/applications), puis sélectionnez **Créer une application**.
2. Saisissez un nom descriptif pour votre application, sélectionnez **Applications Web classiques**, et cliquez sur **Créer**.

Une fois la nouvelle application créée, vous serez dirigé vers **Application****Paramètres**, qui comprend les onglets suivants : 
Pour en savoir plus, consultez [Paramètres de l’application](/applications/application-settings).
3. Faites défiler vers le bas et localisez le paramètre **Faire confiance à l'IP indiqué dans l'entête du point de terminaison du jeton**, activez-le et cliquez sur **Save Changes (Enregistrer les modifications)**. Cette option protège contre les attaques par force brute.
