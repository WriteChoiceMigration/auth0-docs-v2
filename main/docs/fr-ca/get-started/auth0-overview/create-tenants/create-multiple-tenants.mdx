Vous pouvez configurer plusieurs locataires pour créer [différents environnements](/get-started/create-tenants/set-up-multiple-environments) dans Auth0 Dashboard afin de permettre des configurations complexes. Par exemple, vous pouvez avoir deux domaines distincts (un domaine interne et un domaine public), ou vous pourriez souhaiter que les utilisateurs se connectent différemment pour différentes applications. Pour y parvenir, vous pouvez créer plusieurs locataires Auth0 afin de disposer d’ensembles distincts d’applications, de connexions et d’utilisateurs pour les applications et les groupes d’utilisateurs que vous devez prendre en charge.

1. Rendez-vous dans [Auth0 Dashboard](%24%7Bmanage_url%7D/), sélectionnez votre nom de locataire et sélectionnez **Créer un locataire**.
2. Saisissez le **Tenant Domain (Domaine locataire)** souhaité, sélectionnez une **Région**, puis cliquez sur **Create (Créer)**.
