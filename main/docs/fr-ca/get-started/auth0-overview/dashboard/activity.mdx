La page Activité fournit un résumé des données clés de votre locataire Auth0, y compris des informations sur les utilisateurs actifs, les échecs de connexion, et plus encore.

## Choses à savoir

* L’affichage de la page Activité d’un locataire nécessite un compte avec un rôle d’administrateur.
* Le rôle **Accès au soutien** a un accès limité à la page Activité du locataire.
* Pour garantir la cohérence des données, la date actuelle ne peut pas être sélectionnée dans le sélecteur de date et l’affichage des données peut prendre jusqu’à 12 à 24 heures.

## Mesures

Voici ce que vous verrez sur la page Activité de votre locataire.

### Totaux

En haut de la page, vous verrez le nombre total d’utilisateurs, d’applications, d’API et de connexions de votre locataire.

<Frame>![Example Dashboard Activity page tenant totals](/images/cdy7uua7fh8z/4wcwZDIhDQVD0vhLxg6k49/b5d36a70ad9a8bdbe8e83b984053ee99/dashboard-activity-totals.png)</Frame>

### Utilisateurs actifs quotidiens

Le nombre d’utilisateurs uniques quotidiens avec une activité d’authentification ou d’autorisation réussie. Vous pouvez voir le nombre d’utilisateurs uniques pour chaque jour en passant votre souris sur un point du graphique.

<Frame>![Example Dashboard Activity page Active Users line graph](/images/cdy7uua7fh8z/Ogdesg0DscG5NhPAQ4zS0/9e8d93b1e65c51b76490cda264db01c7/active_users.png)</Frame>

### Rétention des utilisateurs

Le pourcentage d’utilisateurs actifs au cours de la période donnée, calculé à partir du nombre d’utilisateurs actifs sur le nombre total d’utilisateurs liés au locataire.

<Frame>![Example Dashboard Activity page Rétention des utilisateurs line graph](/images/cdy7uua7fh8z/4Fm2nJoZl9yOpw6Siy5Vem/09cc8adb31ad0905456228551c0a9bc6/user_retention.png)</Frame>

### Inscriptions

Le nombre d’inscriptions complétées avec succès.

<Frame>![Example Dashboard Activity page Sign ups line graph](/images/cdy7uua7fh8z/y9guAxVBfFaaKIEGwCGRI/9b1a933e2da2d7a4cf168892b5fbdc35/sign_ups.png)</Frame>

### Échecs de connexion

Le nombre d’échecs de connexion de l’utilisateur (le type d’événement `f` du journal) sur la période donnée.

<Frame>![Example Dashboard Activity page Échecs de connexion line graph](/images/cdy7uua7fh8z/35WccBqKhEd5z0uT4axrnZ/c964fdf97e1a33d298fa8906a77bef9c/failed_logins.png)</Frame>

## Comparer à la période précédente

Lorsque **Comparer à la dernière période** est activé :

* La différence en pourcentage est incluse dans le graphique. Les changements positifs sont de couleur verte tandis que les changements négatifs sont de couleur rouge.
* Les lignes pointillées sont la dernière période de temps et les lignes pleines sont la période de temps actuelle.

<Frame>![Example of mouse hovering over the Dashboard Activity page Active Users line graph](/images/cdy7uua7fh8z/3wbi2SCiWbI5JisOxMLD2Z/892abde733b368efd903d68d1d501c10/2025-02-26_14-01-31.png)</Frame>

## Afficher les données par période

Vous pouvez afficher les données pour une période spécifique en utilisant le sélecteur de dates à côté de la case à cocher **Comparer à la période précédente**. Il existe des plages de dates prédéfinies pour les 7/14/30/60 derniers jours, ou vous pouvez fournir les dates **De** et **À** pour sélectionner une plage personnalisée.

<Frame>![Example of Dashboard Activity page datepicker form](/images/cdy7uua7fh8z/4ctRxO704Ufg3q5xn8riCi/e3b174f34a4b9e12f9321a975a10f6c5/date_picker.png)</Frame>
