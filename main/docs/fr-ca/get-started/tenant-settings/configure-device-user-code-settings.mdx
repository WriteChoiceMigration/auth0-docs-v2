Vous pouvez configurer les paramètres du code utilisateur généré par votre application au cours du [Flux d’autorisation d’appareils](/flows/concepts/device-auth) à l’aide d’Auth0 Dashboard.

1. Allez dans [Dashboard (Tableau de Bord) > Settings (Paramètres)](%24%7Bmanage_url%7D/#/tenant) et cliquez sur l’onglet **Advanced (Avancé)**.
2. Faites défiler jusqu’à la section **Format du code utilisateur du flux d’appareils** , localisez **Jeu de caractères du code utilisateur** et **Masque du code utilisateur**, saisissez les paramètres souhaités et cliquez sur **Enregistrer**.
