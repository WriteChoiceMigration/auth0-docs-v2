Chaque locataire Auth0 a un nom. Il s’agit du nom que vous utilisez fréquemment dans votre code pour identifier le locataire. Lorsque vous voyez `{yourTenantId}`, `{yourTenantName}`, `{yourTenant}` ou une variable similaire dans la documentation Auth0, remplacez la variable par le nom du locataire.

1. Accédez à [Auth0 Dashboard](%24%7Bmanage_url%7D).
2. Dans le coin supérieur gauche, vous trouverez le nom du locataire à côté du logo Auth0.
