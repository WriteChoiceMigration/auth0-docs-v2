Dans le [Scénario d’architecture Serveur + API](https://auth0.com/docs/architecture-scenarios/application/server-api), nous avons exploré un scénario d’utilisation simple de machine à machine. Ce scénario implique une API utilisée pour importer des feuilles de temps dans les systèmes d’ExampleCo, ainsi qu’une tâche cron exploitée par des sous-traitants externes pour envoyer leurs feuilles de temps via cette API.

Nous avons appris comment octroyer des identifiants client et des jetons d’accès, comment configurer une API dans Auth0, comment configurer une application machine-machine pour communiquer en toute sécurité avec cette API, comment définir et sécuriser nos points de terminaison API, comment utiliser les bibliothèques fournies pour valider le jeton d’accès et comment en récupérer un nouveau auprès d’Auth0.

Nous espérons que ces exemples vous aideront à développer votre propre implémentation en utilisant vos technologies préférées.
