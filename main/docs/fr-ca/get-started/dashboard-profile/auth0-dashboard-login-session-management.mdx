Avec les sessions de connexion, les administrateurs d’Auth0 Dashboard peuvent suivre les sessions actives de différentes applications d’Auth0 Dashboard sur plusieurs appareils et navigateurs. Vous pouvez examiner et révoquer les sessions créées par les utilisateurs accédant respectivement à Teams Dashboard, au Centre de support et à Auth0 Dashboard.

Vous pouvez voir une liste de toutes les sessions actives sur la [page de profil](https://manage.auth0.com/#/profile/sessions) de votre tableau de bord Auth0.

Pour se déconnecter ou révoquer une session, localisez la session concernée et sélectionnez **Revoke (Révoquer)**.
