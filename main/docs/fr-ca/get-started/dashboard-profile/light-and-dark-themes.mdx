Les clients Auth0 peuvent utiliser le mode clair ou le mode foncé dans leur Dashboard de locataire. Vous pouvez également régler votre Dashboard pour qu’il reproduise les réglages de votre environnement système local.

Il est possible de modifier le thème du Dashboard de deux façons :

1. Cliquer sur le menu déroulant du profil dans le coin supérieur droit de votre Dashboard.
2. Utilisez le menu pour basculer d’un thème à l’autre.

Vous pouvez aussi cliquer **Votre profil** dans le menu déroulant et faire défiler jusqu’en bas, afin de choisir entre les trois thèmes.
