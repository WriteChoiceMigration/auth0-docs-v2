Les clients Auth0 peuvent utiliser le mode clair ou le mode foncé dans leur Dashboard de locataire. Vous pouvez également régler votre Dashboard pour qu’il reproduise les réglages de votre environnement système local.

Il est possible de modifier le thème du Dashboard de deux façons :

1. Select the profile dropdown menu at the top-right of your Dashboard.
2. Use the menu to switch between themes.

<Frame>![Where to find the Dark mode and Light mode toggle](/images/cdy7uua7fh8z/1BImxFvHXoULQVTkqCsQL0/a37f6cfdfce935fd20731aa68f8067de/2024-09-10_09-08-27.png)</Frame>

You may also click **Your Profile** in the same dropdown and scroll to the bottom, where you can choose from the three themes.
