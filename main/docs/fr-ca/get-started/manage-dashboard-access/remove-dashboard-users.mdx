Si vous êtes administrateur locataire, vous pouvez révoquer l’accès au Dashboard.

1. Allez dans [Dashboard > Paramètres > Membres locataires](%24%7Bmanage_url%7D/#/tenant/admins).
2. Cliquez sur **…** à droite du nom ou de l’adresse courriel d’un utilisateur et cliquez sur **Supprimer**.
3. Cliquez sur **Enregistrer**
