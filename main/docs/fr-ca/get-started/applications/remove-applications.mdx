Vous pouvez supprimer une application à partir de l’Auth0 Dashboard ou de Management API. Une fois confirmée, cette opération ne peut être annulée.

## Utiliser le Dashboard

1. Rendez-vous à [Dashboard > Applications > Applications](%24%7Bmanage_url%7D/#/applications), et sélectionnez le nom de l’application que vous souhaitez visualiser.
2. Faites défiler en bas de la page **Paramètres**, localisez la **Zone danger**, sélectionnez **Supprimer**, puis confirmez.

## Utiliser Management API

Faites une requête `DELETE` aux points de terminaison [`/Clients/delete_clients_by_id`](/api/management/v2#!/Clients/delete_clients_by_id). Assurez-vous de remplacer les valeurs des espaces réservés `YOUR_CLIENT_ID` et `MGMT_API_ACCESS_TOKEN` avec votre ID client et votre jeton d’accès de Management API, respectivement.
