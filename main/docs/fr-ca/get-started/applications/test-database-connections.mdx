Vous pouvez tester des connexions de base de données pour les applications utilisant Auth0 Dashboard. Ces connexions configurées peuvent être utilisées pour se connecter à votre application.

Pour tester correctement, vous devez déjà avoir [configuré votre connexion de base de données](/applications/set-up-database-connections) et [créé un utilisateur](/users/create-users) pour votre connexion de base de données.

1. Accédez à [Auth0 Dashboard > Authentification > Base de données](%24%7Bmanage_url%7D/#/connections/database), puis sélectionnez la flèche **Essayer** à côté de la connexion que vous souhaitez tester.
