Vous pouvez utiliser Management API Auth0 pour vérifier si une application est enregistrée avec Auth0 comme application première partie ou comme application tierce partie.

Effectuez une requête `GET` au point de terminaison [Récupérer un client](/api/management/v2#!/Clients/get_clients_by_id). Assurez-vous de remplacer les valeurs des espaces réservés `{yourClientId}`
 et `{yourMgmtApiAccessToken}` avec votre ID client et votre jeton d’accès de Management API, respectivement.





Si votre application est première partie, le champ `is_first_party` aura une valeur de `true`. Si votre application est tierce partie, le champ `is_first_party` aura une valeur de `false`.
