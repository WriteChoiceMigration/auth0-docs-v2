You can change your application’s enabled connections using the Auth0 Dashboard.

1. Rendez-vous à [Dashboard > Applications](%24%7Bmanage_url%7D/#/applications) et cliquez sur le nom de l’application à afficher.
2. Cliquez sur l’onglet **Connexions**, puis activez ou désactivez les connexions appropriées pour l’application.
