Votre plan d’abonnement Auth0 détermine les limites d’utilisateurs et les fonctionnalités disponibles.

## Utilisation des quotas

You can view subscription utilization information from Auth0 Support Center by visiting [Reports > Quota Utilization](https://support.auth0.com/reports/quota). The subscription utilization information is available with the following details:

* Consommation des utilisateurs

  + Utilisateurs actifs
* Consommation de fonctionnalités

  + Connexions d’entreprise
  + Authentification par communication entre machines
  + Authentification multifacteur (MFA)

<Frame>![Reports showing subscription feature usage](/images/cdy7uua7fh8z/7g0PDAKl5D8YEf4vC5FjSI/188182aabbeb7e0af77b1bc9d292cfad/Screenshot_2023-02-07_at_5.35.40_PM.jpeg)</Frame>

Chaque carte récapitulative à niveau élevé montre l’utilisation du mois en cours par rapport à la limite d’abonnement disponible. Il est également possible de zoomer (en cliquant sur la carte) pour afficher des détails supplémentaires tels que la répartition de l’utilisation par Instance privée Auth0 (le cas échéant) et par Locataire pour les 12 derniers mois.

Vous pouvez également télécharger un fichier CSV pour exporter ces données si nécessaire.

<Frame>![](/images/cdy7uua7fh8z/7bBqDuycQWsnUdd9hlXN10/040a04d458bff06e8392c1432e3308ce/Screenshot_2023-02-07_at_6.07.01_PM.jpeg)</Frame>

### Utilisateurs actifs

Vous pouvez l’utiliser pour obtenir le nombre d’utilisateurs uniques ayant un événement actif (connexion ou inscription) au cours d’un mois civil donné. Il est possible de détailler et d’obtenir une répartition de ces utilisateurs actifs par locataire et par instance Auth0 privée (le cas échéant) sur les 12 derniers mois au maximum.

### Connexions d’entreprise

An [Enterprise connection](/docs/authenticate/enterprise-connections) is considered active if (during the current month) it has both:

* a été activée pour accéder à une application
* est associée à des activités (par exemple, une connexion, une inscription ou l’échange de jetons).

Si une connexion d’entreprise n’a jamais été activée pour aucune application, ou si elle a été activée mais n’a fait l’objet d’aucune activité par les utilisateurs pendant le mois en cours, elle n’est pas considérée comme active.

Vous pouvez utiliser ce rapport pour obtenir un décompte des connexions d’entreprise actives pour un mois civil donné avec une répartition des connexions actives par instance Auth0 privée (le cas échéant) ainsi qu’un décompte des utilisateurs actifs par connexion d’entreprise active et par locataire.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

The Okta Workforce Enterprise connection is not counted toward your active Connexions d’entreprise limit.

</Callout>

### Authentification par communication entre machines

This report provides a count of <Tooltip tip="Access Token: Authorization credential, in the form of an opaque string or JWT, used to access an API." cta="View Glossary" href="/docs/glossary?term=access+tokens">access tokens</Tooltip> issued for the [Client Credentials Flow](/docs/get-started/authentication-and-authorization-flow/client-credentials-flow), also referred to as API Authentication Calls, per calendar month excluding any tokens issued for Auth0 <Tooltip tip="Management API: A product to allow customers to perform administrative tasks." cta="View Glossary" href="/docs/glossary?term=Management+API">Management API</Tooltip>.

Vous pouvez consulter le nombre d’appels d’authentification par instance Auth0 privée (le cas échéant) et par locataire.

Pour les locataires qui utilisent l’option de déploiement dans le nuage public d’Auth0, le rapport fournit également une répartition des appels d’authentification par application de locataire pour les 7 derniers jours.

### Authentification multifacteur

There are two reports for Authentification multifacteur (<Tooltip tip="Multi-factor authentication (MFA): User authentication process that uses a factor in addition to username and password such as a code via SMS." cta="View Glossary" href="/docs/glossary?term=MFA">MFA</Tooltip>) usage which present a count of MFA events (enrolments or challenges) by calendar month.

Les décomptes MFA Enterprise sont des événements MFA liés aux notifications poussées, aux WebAuthn et aux codes à usage unique générés par SMS, courriel ou téléphone. Les décomptes MFA Pro sont des événements MFA liés à des applications à code unique telles que Google Authentication ou Cisco DUO.

Both these report types provide a breakdown of MFA counts by Auth0 Private Instance\* (if applicable) and Tenant.

\*Auth0 Private Instance MFA usage is only available since 1st January 2023.

## Utilisation des locataires

You can view total and active user counts by calendar month from Auth0 Support Center for a given tenant by visiting [Reports > Usage](https://support.auth0.com/reports/usage), and selecting a Subscription (and a Tenant for self-service subscriptions). This would provide the following breakdown:

* Active Users: Count of active users by month and user type (Social, Password, <Tooltip tip="Passwordless: Form of authentication that does not rely on a password as the first factor." cta="View Glossary" href="/docs/glossary?term=Passwordless">Passwordless</Tooltip>, Enterprise) along with a Total Active count.
* Utilisateurs actifs par locataire : Nombre d’utilisateurs actifs par mois, par locataire et par type d’utilisateur (réseau social, avec mot de passe, sans mot de passe, entreprise), ainsi que le nombre total d’utilisateurs actifs.
* Nombre total d’utilisateurs par locataire

<Frame>![Support Center -> Reports -> Usage](/images/cdy7uua7fh8z/2RmWp1ZIWwV2JCPJcftll5/acd9c3c71b8dbd9cca12a44588e015af/Screen_Shot_2022-10-27_at_1.21.35_PM.png)</Frame>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

Counts of Total Active or Total Users can be higher than the sum of users by each individual type for situations where user type is not determined. This can happen, but is not limited to, scenarios where account linking is turned on.

</Callout>
