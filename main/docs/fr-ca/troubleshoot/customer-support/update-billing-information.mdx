Vous pouvez utiliser Auth0 Dashboard pour mettre à jour vos informations de facturation. Cette fonction n’est disponible que pour les administrateurs de locataires sur tous les abonnements basés sur les locataires. Pour les abonnements basés sur un compte, vous devez être propriétaire d’équipe pour accéder à cette fonctionnalité.

## Paiement et facturation

Pour mettre à jour vos informations de facturation :

1. Allez à [Auth0 Dashboard > Settings(Paramètres)](%24%7Bmanage_url%7D/#/dashboard/settings).
2. Dans **Tenant Settings (Paramètres du locataire)**,** **sélectionnez **Payment & Billing (Paiement et facturation)**.
3. Cliquez sur **Add Payment Details (Ajouter des détails de paiement )** pour saisir vos informations de paiement.

4. Cliquez sur **Update (Mettre à jour)** pour enregistrer vos modifications.
