* Assurez-vous que le domaine est le même que celui utilisé pendant l’authentification.
* SI vous utilisez un [domaine personnalisé](/custom-domains) Auth0, il est important d’utiliser le même domaine que celui utilisé dans l’application pour invoquer l’authentification. Consultez [Dépannage des domaines personnalisés](/troubleshoot/troubleshoot-integration-and-extensibility/troubleshoot-custom-domains) pour en savoir plus.
