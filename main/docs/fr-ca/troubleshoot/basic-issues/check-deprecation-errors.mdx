En cas de fonctionnalités d’Auth0 devenues obsolètes, il peut y avoir des erreurs ou des avis dans les journaux des locataires qui s’affichent pour indiquer que vos applications utilisent des fonctionnalités obsolètes. Ce guide vous aidera à rechercher dans vos journaux les messages liés à l’obsolescence, ainsi que des explications sur les causes potentielles et les solutions pour des éléments particuliers.

There are two different ways to search for warning messages showing usage of deprecated features: The Dashboard or the <Tooltip tip="Management API: A product to allow customers to perform administrative tasks." cta="View Glossary" href="/docs/glossary?term=Management+API">Management API</Tooltip>. Note that in either case, the [log retention period](/docs/deploy-monitor/logs) is governed by the subscription level of your account.

See [Search Logs for Deprecation Errors](/docs/troubleshoot/basic-issues/search-logs-for-deprecation-errors) for details.

## Messages du journal d’obsolescence

### Legacy Lock API

**Entrée de journal**: `up-idp-initiated`

**Message d'erreur:** Legacy Lock API : Cette fonctionnalité est déconseillée. Veuillez vous reporter à notre documentation pour savoir comment migrer votre application

**Cause**: Les entrées du journal des locataires relatives à l’API Legacy Lock peuvent contenir des informations sur le référent ainsi que sur la trousse SDK utilisée.

**Résolution** : Servez-vous de ces informations pour vérifier si l’une de vos applications repose sur des bibliothèques obsolètes.

### Point de terminaison SSOdata

**Entrée de journal**: `ssodata`

**Message d'erreur:** Point de terminaison SSOdata : Cette fonctionnalité est déconseillée. Veuillez vous reporter à notre documentation pour savoir comment migrer votre application.

**Cause**: Soit en appelant le point de terminaison `/ssodata` directement ou en utilisant d’anciennes versions de la trousse SDK Lock ou Auth0.js intégrée pour appeler une fonction qui a appelé le point de terminaison `/ssodata`.

**Résolution**: [Migrer vers la connexion universelle](/not-found/5I5soJeKPA2wfbpfkHceZy)
