## Limites

Auth0 limite le nombre total d’utilisateurs que vous pouvez récupérer à 1 000 (p. ex., 100 utilisateurs par page pour 10 pages). Si le paramètre `page` n’est pas défini, nous renvoyons un maximum de 50 enregistrements, quel que soit le nombre d’enregistrements existants.
