Pour trier les résultats de recherche, passez une valeur `field:order` au paramètre `sort` lorsque vous effectuez votre requête. Le `field` est le nom du champ de tri, alors que order peut être défini sur `1` pour un ordre croissant et `-1` pour un ordre décroissant.

Le tri par `app_metadata` ou `user_metadata` n’est pas pris en charge.
