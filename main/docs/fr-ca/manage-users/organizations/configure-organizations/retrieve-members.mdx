Lorsque vous travaillez avec des [organisations](/organizations/organizations-overview) de manière programmatique, vous pouvez avoir besoin de récupérer la liste des membres d’organisation.

Vous pouvez visualiser les membres associés à une organisation en utilisant Auth0 Dashboard ou Management API.

## Auth0 Dashboard

1. Naviguez vers [Auth0 Dashboard > Organizations](%24%7Bmanage_url%7D/#/organizations), et sélectionnez l’organisation pour laquelle vous souhaitez voir les membres.
2. Sélectionnez la vue **Membres**.

## Management API



Effectuez un appel `GET` vers le point de terminaison `Obtenir les utilisateurs de l’organisation`. Assurez-vous de remplacer les valeurs des paramètres fictifs `ORG_ID` et `MGMT_API_ACCESS_TOKEN` par l’identifiant de votre organisation et le jeton d’accès à Management API, respectivement.
