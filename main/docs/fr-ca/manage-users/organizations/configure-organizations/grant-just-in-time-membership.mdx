L’autorisation d’une adhésion JIT permet aux utilisateurs qui se connectent via une connexion activée de se voir automatiquement attribuer une adhésion à une [organisation](/organizations/organizations-overview).

Pour accorder automatiquement une adhésion aux utilisateurs lorsqu’ils se connectent à une organisation, vous devez d’abord [configurer et activer une connexion](/organizations/enable-connections) pour cette organisation. Lors de l’activation de votre connexion, sélectionnez **Activer l’abonnement automatique** pour permettre à tous les utilisateurs se connectant avec la connexion d’être automatiquement ajoutés en tant que membres de cette organisation.
