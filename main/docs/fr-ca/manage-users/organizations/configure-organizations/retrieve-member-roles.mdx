Lorsque vous travaillez programmatiquement avec des [organisations](/organizations/organizations-overview), vous devrez peut-être récupérer une liste des rôles attribués à un membre d’une organisation.

Vous pouvez afficher les rôles associés à un membre de l’organisation à l’aide du Auth0 Dashboard (Tableau de bord Auth0) ou de Management API.

## Auth0 Dashboard

1. Naviguez vers [Auth0 Dashboard > Organizations](%24%7Bmanage_url%7D/#/organizations), et sélectionnez l’organisation pour laquelle vous souhaitez voir les membres.
2. Sélectionnez la vue **Membres**, puis sélectionnez l’utilisateur pour lequel vous souhaitez afficher les rôles.

## Management API

Effectuez une requête `GET` au point de terminaison `Obtenir les rôles des membres d'une organisation`. Assurez-vous de remplacer les valeurs des espaces réservés `ORG_ID`, `USER_ID`, and `MGMT_API_ACCESS_TOKEN`, avec votre ID d’organisation, l’ID d’utilisateur du membre et votre jeton d’accès du Management API, respectivement.
