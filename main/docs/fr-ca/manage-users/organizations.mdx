La fonctionnalité Organisations représente une vaste mise à jour de la plateforme Auth0 qui permet à nos clients interentreprises (B2B) de mieux gérer leurs partenaires et leurs clients, et de personnaliser les modes d’accès des utilisateurs finaux à leurs applications. Les clients Auth0 peuvent utiliser Organizations pour :

* représenter leurs clients et partenaires dans Auth0 et gérer leur adhésion;
* configurer des flux de connexion fédérés pour chaque entreprise;
* offrir un accès de communication entre machines à leurs clients et partenaires professionnels pour consommer leurs API B2B.
* intégrer des capacités d’administration dans leurs produits, en utilisant les API d’Organizations, afin que ces entreprises puissent gérer leurs propres organisations.
