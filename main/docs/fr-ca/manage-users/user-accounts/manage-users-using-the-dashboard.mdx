Le Dashboard vous permet de gérer vos utilisateurs. Pour commencer, rendez-vous à [Dashboard > Gestion des utilisateurs > Utilisateurs](%24%7Bmanage_url%7D/#/users). Vous pourrez alors créer, afficher, modifier et supprimer des utilisateurs.
