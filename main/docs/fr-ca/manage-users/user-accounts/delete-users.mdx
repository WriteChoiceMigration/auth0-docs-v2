1. Allez à [Dashboard > Gestion des utilisateurs - Utilisateurs](%24%7Bmanage_url%7D/#/users).
2. Sélectionnez le nom de l’utilisateur que vous souhaitez supprimer.
3. Dans l’onglet **Détails**, faites défiler vers le bas et, à côté de **Supprimer l’utilisateur**, cliquez sur **Supprimer**.

Un message s’affichera pour vous avertir que l’action est irréversible et vous demandera de confirmer la suppression de l’utilisateur. Cliquez sur le bouton **Oui, le supprimer** pour confirmer. Cela supprimera de façon permanente l’utilisateur.
