Vous pouvez établir les attributs racine pour les utilisateurs au cours de l’inscription en utilisant Management API Auth0. Cela permet de minimiser le nombre de requêtes API nécessaires pour établir les attributs racines lorsque vous créez des utilisateurs.

Faites une requête `POST` au [point de terminaison Créer un utilisateur](/api/management/v2#!/Users/post_users). Assurez-vous de remplacer les espaces réservés `MGMT_API_ACCESS_TOKEN`, `CONNECTION_NAME`, `EMAIL_VALUE`, `PASSWORD_VALUE`, `GIVEN_NAME_VALUE`, `FAMILY_NAME_VALUE`, `NAME_VALUE`, `NICKNAME_VALUE`, and `PICTURE` avec votre jeton d’accès à Management API, votre nom de connexion initial, votre adresse courriel, votre mot de passe, votre prénom, votre nom de famille, votre nom, votre surnom et votre photo, URL, respectivement.
