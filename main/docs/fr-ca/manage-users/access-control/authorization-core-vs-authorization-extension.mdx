Auth0 propose actuellement deux façons de mettre en œuvre le [contrôle d’accès basé sur les rôles (RBAC)](/rbac) : notre implémentation de base et notre Authorization Extension. Notre implémentation de base améliore les performances et l’extensibilité.

Nous vous recommandons d’utiliser Authorization Core pour la plupart des implémentations. Si vous souhaitez représenter des équipes, des clients professionnels ou des partenaires dans une application inter-entrprises ou SaaS, nous vous recommandons de les représenter en tant qu’[organisations](/organizations/organizations-overview) et d’utiliser Authorization Core. Authorization Extension ne prend pas en charge les organisations.
