Les fonctionnalités Authorization Core d’Auth0 permettent un contrôle d'accès basé sur les rôles (RBAC) de vos API.

Pour que le RBAC fonctionne correctement, vous devez l’activer pour votre API à l’aide du Tableau de bord  ou de Management API. Pour en savoir plus, lisez [Activer le contrôle d'accès basé sur les rôles (RBAC) pour les API](/apis/enable-role-based-access-control-for-apis).

La fonctionnalité Authorization Core est différente de Authorization Extension (Extension d’autorisation). Pour une comparaison entre les deux produits, lisez [Authorization Core par rapport à Authorization Extension](/rbac/authorization-core-vs-authorization-extension).

Pour utiliser la fonctionnalité de base le plus efficacement possible, vous devez procéder comme suit :

1. [Enregistrer l’API auprès d’Auth0](/get-started/set-up-apis)
2. [Définir les autorisations pour l’API](/apis/add-api-permissions)
3. [Créer des rôles](/rbac/roles/create-roles)
4. [Attribuer des rôles aux utilisateurs](/rbac/rbac-users/assign-roles-to-users)
5. [Attribuer des autorisations aux utilisateurs](/rbac/rbac-users/assign-permissions-to-users), le cas échéant.
