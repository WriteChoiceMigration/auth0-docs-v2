Vous pouvez gérer les utilisateurs dans un système de contrôle d'accès basé sur les rôles (RBAC).

Nous offrons une variété de fonctionnalités pour faciliter la gestion de vos utilisateurs, accessibles depuis Auth0 Dashboard et Auth0 Management API :

* [Attribution de rôles aux membres](/rbac/rbac-users/assign-roles-to-users)
* [Afficher les rôles attribués aux utilisateurs](/rbac/rbac-users/view-user-roles)
* [Supprimer les rôles des utilisateurs](/rbac/rbac-users/remove-roles-from-users)
* [Attribuez des autorisations aux utilisateurs](/rbac/rbac-users/assign-permissions-to-users)
* [Afficher les autorisations d’un utilisateur](/rbac/rbac-users/view-user-permissions)
* [Supprimer les autorisations des utilisateurs](/rbac/rbac-users/remove-permissions-from-users)
