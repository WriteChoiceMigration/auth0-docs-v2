Vous pouvez afficher les utilisateurs affectés à un [rôle](/authorization/concepts/rbac) en utilisant Auth0 Dashboard ou Management API. Les rôles sont utilisés avec l’ensemble de fonctionnalités API d’Authorization Core.

## Prérequis



## Dashboard

1. Allez à [Dashboard > Gestion des utilisateurs > Rôles](%24%7Bmanage_url%7D/#/users) et cliquez sur le nom du rôle à afficher.
2. Cliquez sur l’onglet **Utilisateurs**.

Les informations suivantes sont affichées pour chaque utilisateur :



## Management API

To retrieve a list of users assigned to a role, see [Get a Role’s Users](https://auth0.com/docs/api/management/v2#!/Roles/get_role_user).
