Vous pouvez modifier un [rôle](/rbac) avec Auth0 Dashboard ou Management API. Les rôles sont utilisés avec l’ensemble de fonctionnalités API Authorization Core.

## Conditions préalables



## Dashboard

1. Rendez-vous dans [Dashboard > Gestion des utilisateurs > Rôles](%24%7Bmanage_url%7D/#/roles) et cliquez sur le nom du rôle à afficher.
2. Modifiez le nom et la description du rôle, puis cliquez sur **Enregistrer**.

## Management API

Faites un appel `PATCH` au point de terminaison [Mettre à jour le rôle](/api/management/v2#!/Roles/patch_roles_by_id). Assurez-vous de remplacer les valeurs des placeholders `ROLE_ID`, `MGMT_API_ACCESS_TOKEN`, `ROLE_NAME`, et `ROLE_DESC` par votre identifiant de rôle, votre jeton d’accès à Management API, le nom et la description de votre rôle, respectivement.
