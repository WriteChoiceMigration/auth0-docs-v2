Vous pouvez créer des [rôles](/rbac) à l'aide d'Auth0 Dashboard ou de Management API. Les rôles peuvent être utilisés avec l'ensemble de fonctionnalités API Authorization Core.

## Prérequis

* 
* [Mettre en place une API](/get-started/set-up-apis) dans l’Auth0 Dashboard.
* Les autorisations sont sélectionnées parmi des valeurs prédéfinies. Si votre liste d’autorisations est vide, vous devez [ajouter des autorisations](/apis/add-api-permissions) à votre API.

## Dashboard

1. Accédez à [Dashboard > Gestion des utilisateurs > Rôles](%24%7Bmanage_url%7D/#/roles) et cliquez sur**Créer un rôle**.
2. Nommez le rôle et ajoutez une description, puis cliquez sur **Créer**.

## Management API

Faites un appel `POST` au [point de terminaison Créer un rôle](https://auth0.com/docs/api/management/v2#!/Roles/post_roles). Assurez-vous de remplacer les valeurs des paramètres fictifs `MGMT_API_ACCESS_TOKEN`, `ROLE_NAME` et `ROLE_DESC` par votre jeton d’accès à Management API, le nom et la description de votre rôle, respectivement.
