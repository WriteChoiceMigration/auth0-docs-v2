Vous pouvez afficher les [autorisations](/rbac) attribuées à un rôle en utilisant Auth0 Dashboard ou Management API. Les autorisations et rôles attribués sont utilisés avec l’ensemble de fonctionnalités de Authorization Core API.

## Prérequis



## Dashboard

1. Allez à [Dashboard > Gestion des utilisateurs > Rôles](%24%7Bmanage_url%7D/#/roles) et cliquez sur le nom du rôle à afficher.
2. Cliquez sur l’affichage **Autorisations**.

Les informations suivantes sont affichées pour chaque autorisation :

## Management API

Faites un appel `GET` au [point de terminaison Obtenir les autorisations des rôles](https://auth0.com/docs/api/management/v2#!/Roles/get_role_permission). Assurez-vous de remplacer les valeurs des paramètres fictifs `ROLE_ID` et `MGMT_API_ACCESS_TOKEN` avec votre identifiant de rôle et votre jeton d’accès de Management API, respectivement.
