Vous pouvez supprimer un [rôle](/rbac) avec Auth0 Dashboard ou Management API. Les rôles sont utilisés avec l’ensemble de fonctionnalités API Authorization Core.



## Dashboard

1. Allez à [Dashboard > Gestion des utilisateurs > Rôles](%24%7Bmanage_url%7D/#/roles) et cliquez sur le nom du rôle à afficher.
2. Cliquez sur **Supprimer ce rôle**, puis confirmez.

## Management API

Faites un appel `DELETE` au [point de terminaison Supprimer le rôle](https://auth0.com/docs/api/management/v2#!/Roles/delete_roles_by_id). Assurez-vous de remplacer les valeurs des paramètres fictifs `ROLE_ID` et `MGMT_API_ACCESS_TOKEN` avec votre identifiant de rôle et votre jeton d’accès de Management API, respectivement.
