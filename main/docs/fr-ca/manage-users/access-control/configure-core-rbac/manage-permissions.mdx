Ce guide vous montrera comment gérer les autorisations dans un système de [contrôle d'accès basé sur les rôles (RBAC)](/authorization/concepts/rbac). Ces autorisations sont utilisées avec l’ensemble de fonctionnalités API Authorization Core.

Nous proposons différentes fonctions pour vous aider à gérer vos autorisations, auxquelles vous pouvez accéder à partir d’Auth0 Dashboard ou d’Auth0 Management API.

* [Ajouter des autorisations API](/apis/add-api-permissions)
* [Supprimer des autorisations API](/apis/delete-api-permissions)
* [Ajouter des autorisations aux Rôles](/rbac/roles/add-permissions-to-roles)
* [Attribuez des autorisations aux utilisateurs](/rbac/rbac-users/assign-permissions-to-users)
* [Afficher les autorisations des rôles](/rbac/roles/view-role-permissions)
* [Afficher les autorisations utilisateur](/rbac/rbac-users/view-user-permissions)
* [Supprimer les autorisations de rôle](/rbac/roles/remove-permissions-from-roles)
* [Supprimer les autorisations des utilisateurs](/rbac/rbac-users/remove-permissions-from-users)
