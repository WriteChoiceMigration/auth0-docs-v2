You can view the [permissions](/rbac) assigned to a user using the Auth0 Dashboard. Cette tâche peut également être effectuée à l’aide de Management API. Les autorisations attribuées sont utilisées avec l’ensemble de fonctionnalités API Authorization Core.

## Prérequis



## Dashboard

1. Rendez-vous à [Dashboard > Gestion des utilisateurs > Utilisateurs](%24%7Bmanage_url%7D/#/users) et cliquez sur le nom de l’utilisateur à afficher.
2. Cliquez sur l’affichage **Autorisations**.

Les informations suivantes sont affichées pour chaque autorisation :



## Management API

Faites un appel `GET` à votre [point de terminaison Obtenir les permissions de l’utilisateur](https://auth0.com/docs/api/management/v2#!/Users/get_permissions). Assurez-vous de remplacer les valeurs d’espace réservé `USER_ID` et `MGMT_API_ACCESS_TOKEN` par votre identifiant utilisateur et le jeton d’accès à Management API, respectivement.
