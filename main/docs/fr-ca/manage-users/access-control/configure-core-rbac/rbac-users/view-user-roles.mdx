Vous pouvez afficher les [roles](/rbac) (rôles) attribués à un utilisateur en utilisant l’Auth0 Dashboard. Cette tâche peut également être effectuée à l’aide de Management API. Les rôles attribués sont utilisés avec l’ensemble de fonctionnalités API Authorization Core.

## Prérequis



## Dashboard

1. Allez dans [Dashboard > Gestion des utilisateurs > Utilisateurs](%24%7Bmanage_url%7D/#/users) et cliquez sur le nom de l’utilisateur à afficher.
2. Cliquez sur l’onglet **Rôles**.

Les informations suivantes sont affichées pour chaque rôle :



## Management API

Effectuer un appel `GET` au point de terminaison [Get User Roles](https://auth0.com/docs/api/management/v2#!/Users/get_user_roles) (Obtenir les rôles des utilisateurs). Assurez-vous de remplacer les valeurs d’espace réservé `USER_ID` et `MGMT_API_ACCESS_TOKEN` par votre identifiant utilisateur et le jeton d’accès à Management API, respectivement.
