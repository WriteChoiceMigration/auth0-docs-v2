Cette bibliothèque vous offre la possibilité de personnaliser le comportement du client réseau pour des cas d’utilisation courants, ou de définir et utiliser votre propre implémentation de client réseau.

Vous pouvez configurer la classe Auth0 avec un `NetworkingClient`, qui sera utilisé lors des demandes. Vous pouvez configurer le client par défaut avec des délais d’expiration personnalisés, des en-têtes spécifiques à inclure dans toutes les requêtes, ainsi que l’activation ou non de l’enregistrement des informations de requêtes et de réponses (uniquement à des fins de débogage hors production). Pour une configuration plus avancée, vous pouvez fournir votre propre implémentation de `NetworkingClient`.

### [](https://github.com/auth0/Auth0.Android/tree/v2-dev#set-additional-headers-for-all-requests) Définir des en-têtes supplémentaires pour toutes les demandes



### [](https://github.com/auth0/Auth0.Android/tree/v2-dev#advanced-configuration) Configuration avancée

Pour une configuration plus avancée du client réseau, vous pouvez configurer votre propre implémentation de `NetworkingClient`. Cela peut être utile lorsque vous souhaitez réutiliser votre propre client réseau, configurer un proxy, etc.
