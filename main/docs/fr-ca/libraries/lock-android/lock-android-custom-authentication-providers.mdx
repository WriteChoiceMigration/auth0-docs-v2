**Auth0.Android** inclut la classe `WebAuthProvider` pour traiter le flux d’autorisation à l’aide du navigateur. Et si vous voulez utiliser votre propre implémentation ou une version native d’un `AuthProvider`?

## Classe AuthProvider

Créez une classe qui met en œuvre l’interface `AuthProvider` et remplacez ses méthodes.

You can also use any **Native** implementation already provided by Auth0. The [native social authentication](/docs/libraries/lock-android/lock-android-native-social-authentication) providers currently available are Google and Facebook.
