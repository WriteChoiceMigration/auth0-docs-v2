title: Ajouter la connexion à votre application Ionic Vue avec l’application Capacitor
---

Auth0 vous permet d’ajouter rapidement l’authentification et de pouvoir accéder aux informations relatives au profil utilisateur dans votre application. Ce guide montre comment intégrer Auth0 avec une application Ionic (Vue) et Capacitor en utilisant la trousse [SDK Auth0 pour Vue](https://github.com/auth0/auth0-vue).
