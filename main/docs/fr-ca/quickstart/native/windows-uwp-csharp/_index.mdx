Ce tutoriel explique comment ajouter la connexion utilisateur à une application UWP C# à l’aide d’Auth0. Nous vous recommandons de vous connecter pour suivre ce guide rapide avec les exemples configurés pour votre compte.

## Configuration requise

Ce tutoriel et le projet d’exemple ont été testés avec les éléments suivants :

* Microsoft Visual Studio 2022
* Trousse SDK Windows 10 (10.0.26100.0)
* Auth0.OidcClient.UWP 4.0.0
