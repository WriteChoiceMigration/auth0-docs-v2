Ce guide explique comment intégrer Auth0 à n’importe quelle application API Go, nouvelle ou existante, en utilisant le package [go-jwt-middleware](https://github.com/auth0/go-jwt-middleware).

Si vous n’avez pas encore créé d’API dans votre Auth0 Dashboard, vous pouvez utiliser le sélecteur interactif pour créer une nouvelle API Auth0 ou sélectionner une API existante pour votre projet.

Pour configurer votre première API via Auth0 Dashboard, consultez notre [guide de démarrage](/get-started/set-up-apis).

Chaque API Auth0 utilise l’identifiant API, dont votre application a besoin pour valider le jeton d’accès.
