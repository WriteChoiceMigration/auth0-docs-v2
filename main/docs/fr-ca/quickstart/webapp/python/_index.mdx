title: Ajouter une fonctionnalité de connexion à votre application Python Flask
---

Auth0 vous permet d’ajouter l’authentification et de pouvoir accéder aux informations relatives au profil utilisateur dans votre application. Ce guide explique comment intégrer Auth0 à une application Python [Flask](https://flask.palletsprojects.com/) à l’aide de la trousse SDK [Authlib](https://authlib.org/).
