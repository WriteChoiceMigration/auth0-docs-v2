Auth0 Dashboardを使用して、テナントで発生するすべてのイベント（ユーザー認証やアプリケーション、接続、ルールの追加・更新などの管理アクションを含む）を表示します。管理アクションは**API操作**イベントとしてログに表示されます。



1. [［Dashboard］>［Monitoring（モニタリング）］>［Logs（ログ）］](%24%7Bmanage_url%7D/#/logs)に移動します。
2. リストには最新のイベントが表示されます。検索を行うには、カーソルを検索フィールドに置き、クエリを入力して、**Enter**を押します。たとえば、"Pass"を含む接続を使ったすべてのログを検索するには、`connection:*pass*`と入力します。検索クエリ構文と例の詳細については、「[ログ検索クエリ構文](/logs/log-search-query-syntax)」をお読みください。
3. 検索はイベントタイプごとにフィルター処理することができます。ドロップダウンリストからフィルターを選択し、**［Apply（適用）］**をクリックします。たとえば、失敗したユーザーログインイベントにフィルターを適用するには、**［Failed Login（失敗したログイン）］**を選択します。ログイベントフィルターと例の詳細については、「[ログイベントフィルター](/logs/log-event-filters)」をお読みください。
4. 検索で日付範囲を選択することができます。日付フィールドの横のドロップダウンをクリックし、特定の日付または範囲を選択し、**［Apply（適用）］**をクリックします。
