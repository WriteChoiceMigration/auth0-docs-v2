Auth0のログ保持期間はサブスクリプションのレベルによって異なります。



保存期間を超えてログを保持する必要がある場合には、[Auth0サポート](https://support.auth0.com)が、外部のログサービスとの統合をお手伝いします。この方法は、組織全体でログを集約する場合にも役立ちます。



また、Auth0ログストリーミングを使用してログデータをエクスポートすることもできます。詳細については、Auth0 Marketplaceの「[ログストリーミング](https://marketplace.auth0.com/features/log-streaming)」を参照してください。
