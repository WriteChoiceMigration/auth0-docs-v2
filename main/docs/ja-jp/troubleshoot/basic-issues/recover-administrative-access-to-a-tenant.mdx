テナントの管理者アカウントにアクセスできない場合、復旧の方法があります。

既存の管理者が他のチームメンバーにアクセス権を委任せずに組織から抜けた場合、Auth0サポートに問い合わせる前に以下の手順を試してください。



1. 前管理者のメールエイリアスを確認します。Google Workspaceなどを通じてメールの受信箱にアクセスできる場合、該当ユーザーのパスワードリセットを依頼し、テナントアカウントのパスワードをリセットします。
2. テナントの別の管理者について確認します。その管理者が前従業員のアカウントを削除し、新しい管理者を追加することができます。Auth0はテナントに複数の管理者を置くことを推奨しています。

それでもテナントにアクセスできない場合は、Auth0にサポートリクエストを作成してください。サポートチームがテナント所有権を検証し、アクセス権を付与します。詳細については、「[サポートチケットの開示と管理](/support/open-and-manage-support-tickets)」をお読みください。
