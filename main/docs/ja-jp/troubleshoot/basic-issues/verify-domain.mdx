* ドメインが認証中に使用したドメインと同じであることを確認します。
* If using an Auth0 [Custom Domain](/docs/customize/custom-domains), it is important to use the same domain as used in the application to invoke authentication. See [Troubleshoot Custom Domains](/docs/troubleshoot/integration-extensibility-issues/troubleshoot-custom-domains) for details.
