* ドメインが認証中に使用したドメインと同じであることを確認します。
* Auth0[カスタムドメイン](/custom-domains)を使用する場合、認証を実行するためにアプリケーションで使用されているドメインと同じドメインを使用することが重要です。詳細は、「[カスタムドメインのトラブルシューティング](/troubleshoot/troubleshoot-integration-and-extensibility/troubleshoot-custom-domains)」をご覧ください。
