## データのエクスポート

If you would like to export your data from Auth0 there are a several ways you can do this. Please note these tools do not export password hashes of your Auth0-hosted database users. You can still request this information by opening a [support ticket](https://support.auth0.com/). If you are opting to end your use of our service, see [Export Data](/docs/troubleshoot/customer-support/manage-subscriptions/export-data) for details.

### インポート/エクスポート拡張機能の使用

You can use our [Import/Export Extension](/docs/manage-users/user-migration/user-import-export-extension) to export nearly all data in your Auth0 tenant.

### 特定のデータセットをプログラムでエクスポートする場合は、Management APIの一括ユーザーエクスポートエンドポイントを使用します。

If you want to export certain sets of data programmatically, use the <Tooltip tip="Management API: A product to allow customers to perform administrative tasks." cta="View Glossary" href="/docs/glossary?term=Management+API">Management API</Tooltip> Bulk User Export endpoint.

## Auth0は、1つのAuth0テナントから他のAuth0テナントにデータを転送しません。これは、パブリッククラウドとプライベートクラウドの両方の顧客に適用されます。

Auth0テナントのデータはすべて、常にユーザーが管理でき、Management APIからいつでも利用することができます。セキュリティ上の理由から、APIで利用できない情報は、Auth0ホストのデータベースユーザーと秘密鍵のパスワードハッシュのみです。

All data in your Auth0 tenant is always under your control and is available through the Management API at any time. The only information which is not available through the API are the password hashes of your Auth0-hosted database users and private keys, for security reasons.
