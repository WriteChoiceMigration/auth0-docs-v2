* [API呼び出しの確認](/troubleshoot/troubleshoot-authentication/check-api-calls) ｰ API呼び出しを確認して、問題のトラブルシューティング方法を学びます。

* [ユーザープロファイルの確認](/troubleshoot/troubleshoot-authentication/check-user-profiles) ｰ ユーザープロファイルを確認して、問題のトラブルシューティング方法を学びます。

* [SAMLエラーのトラブルシューティング](/troubleshoot/troubleshoot-authentication/saml-errors) - 一般的なSAMLエラーに対するトラブルシューティングについて説明します。
* [セルフパスワード変更のエラー](/troubleshoot/troubleshoot-authentication/self-change-password-errors) ｰ セルフパスワード変更のAPIで発生する可能性のあるエラーコードとその解決策について説明します。
* [認可の拡張機能のトラブルシューティング](/troubleshoot/troubleshoot-authentication/troubleshoot-authorization-extension) - 認可の拡張機能に関するトラブルシューティングについて学びます。
* [Safariを使用する際のトークンの更新に関するトラブルシューティング](/troubleshoot/troubleshoot-authentication/renew-tokens-when-using-safari) - Safariブラウザーの動作と認証に対するトラブルシューティングについて学びます。
