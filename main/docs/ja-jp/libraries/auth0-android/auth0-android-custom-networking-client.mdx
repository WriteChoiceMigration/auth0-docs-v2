このライブラリでは、一般的な構成のネットワーキングクライアントの動作をカスタマイズし、独自のネットワーキングクライアント実装を定義して使用できます。

要求に使用される`NetworkingClient`を用いて、Auth0クラスを構成できます。カスタムタイムアウト値、すべての要求で送信するヘッダー、および要求や応答情報をログに記録するかどうか（非本番環境のデバッグ目的のみ）を使用して、デフォルトのクライアントを構成できます。より高度な構成を行うには、`NetworkingClient`の独自の実装を提供できます。

### Timeout configuration

```kotlin lines
val netClient = DefaultClient(
    connectTimeout = 30,
    readTimeout = 30
)

val account = Auth0("{YOUR_CLIENT_ID}", "{YOUR_DOMAIN}")
account.networkingClient = netClient
```

### Logging configuration

```kotlin lines
val netClient = DefaultClient(
    enableLogging = true
)

val account = Auth0("{YOUR_CLIENT_ID}", "{YOUR_DOMAIN}")
account.networkingClient = netClient
```

### Set additional headers for all requests

```kotlin lines
val netClient = DefaultClient(
    defaultHeaders = mapOf("{HEADER-NAME}" to "{HEADER-VALUE}")
)

val account = Auth0("{YOUR_CLIENT_ID}", "{YOUR_DOMAIN}")
account.networkingClient = netClient
```
