Auth0をネイティブアプリケーションと統合するには、まず、Auth0 Dashboardを使ってアプリをAuth0に登録する必要があります。デバイス（i0S、Androidなど）でネイティブ実行されているモバイル、デスクトップ、ハイブリッドのアプリが含まれます。

1. [［Dashboard］>［Applications（アプリケーション）］>［Applications（アプリケーション）］](%24%7Bmanage_url%7D/#/applications)に移動して、**［Create Application（アプリケーションの作成）］**をクリックします。
2. アプリケーションにわかりやすい名前を付けて、**［Native（ネイティブ）］**を選択してから**［Create（作成）］**をクリックします。

新しいアプリケーションが作成されたら、**［Application ****Settings（アプリケーション設定）］**に移動して、以下のタブを表示します。 
詳細については、「[アプリケーションの設定](/applications/application-settings)」をお読みください。
3. モバイルアプリを開発している場合には、iOSまたはAndroidに必要なパラメーターを**［Advanced Settings（詳細設定）］**に入力してから、**［Save Changes（変更の保存）］**をクリックします。

* iOSアプリには、[**［Team ID（チームID）］**](/applications/enable-universal-links-support-in-apple-xcode)[と](/applications/enable-universal-links-support-in-apple-xcode)[**［App Bundle Identifier（アプリのバンドル識別子）］**](/applications/enable-universal-links-support-in-apple-xcode)を入力します。
* Androidアプリには、[**［App Package Name（アプリのパッケージ名）］**](/applications/enable-android-app-links-support)[と](/applications/enable-android-app-links-support)[**［Key Hashes（キーハッシュ）］**](/applications/enable-android-app-links-support)を入力します。
