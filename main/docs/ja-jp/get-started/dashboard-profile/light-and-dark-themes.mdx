Auth0では、テナントのDashboardでライトモードまたはダークモードを使用できます。また、ローカルシステム環境と同じ設定をDashboardに適用して使うことも可能です。

Dashboardのテーマを変える方法は2つあります。

1. Dashboardの右上にあるプロファイルのドロップダウンメニューをクリックします。
2. メニューを使ってテーマを切り替えます。

あるいは、同じドロップダウンにある**［Your Profile（あなたのプロファイル）］**をクリックしてから、下までスクロールし、表示される3種類のテーマから選択できます。
