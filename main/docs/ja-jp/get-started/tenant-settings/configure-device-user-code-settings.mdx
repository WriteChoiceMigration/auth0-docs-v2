Auth0 Dashboardを用いた[デバイス認可フロー](/flows/concepts/device-auth)中に、アプリが生成したユーザーコードを構成することができます。

1. [［Dashboard］>［Settings（設定）］](%24%7Bmanage_url%7D/#/tenant)に移動して、**［Advanced（詳細設定）］**タブをクリックします。
2. **Device Flow User Code Format（デバイスフローのユーザーコード形式）**セクションまでスクロールし、**ユーザーコード文字セット**と**ユーザーコードマスク**を見つけて、必要な設定を入力し、**［Save（保存）］**をクリックします。
