テナント管理者である場合、割り当てられたユーザーロールを変更することができます。変更は、ユーザーがDashboardをリフレッシュした後に有効になります。

ユーザーには複数のロールを割り当てることができますが、その場合、アクセス権は統合されます。テナントメンバーは、事前に追加されている必要があります。

1. [［Dashboard］>［Settings（設定）］>［Tenant Members（テナントメンバー）］](%24%7Bmanage_url%7D/#/tenant/admins)に移動します。
2. ユーザー名またはメールアドレスの隣にある**［...］**をクリックしてから、**［Edit Roles（ロールの編集）］**をクリックします。このオプションが表示されない場合、そのユーザーが招待をまだ受け入れていないことを意味します。
3. 追加するロールを選択するか、ロールを削除します。
4. **［Save（保存）］**をクリックします。



**パスワードのリセット**：ご自身またはDashboardユーザーの誰かが、ログイン時に問題が発生している場合、詳細については、「[Auth0アカウントパスワードをリセットする](/support/reset-account-passwords)」をお読みください。

**無効なメールアドレス**：テナントメンバーのメールアドレスを表示して、必要に応じて更新するための詳細については、「[Dashboardユーザーのメールアドレスを更新する](/manage-dashboard-access/update-dashboard-user-email)」をお読みください。
