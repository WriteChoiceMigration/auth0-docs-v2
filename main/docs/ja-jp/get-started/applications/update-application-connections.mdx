Auth0 Dashboardを使用して、アプリケーションの有効化された接続を変更できます。

1. [［Dashboard］>［Applications（アプリケーション）］](%24%7Bmanage_url%7D/#/applications)に移動し、アプリケーションの名前をクリックして表示します。
2. **［Connections（接続）］**タブをクリックし、アプリケーションの該当する接続を有効化または無効化します。
