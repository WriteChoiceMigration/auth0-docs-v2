[テナントログ](/logs)を通じて、データベース接続に関連付けられたパスキー フローを監視できます。

Auth0 Dashboard（[Monitoring（監視） > Logs（ログ）](https://manage.auth0.com/#/logs)）でログイベントを表示するか、[Management API](https://auth0.com/docs/api/management/v2/logs/get-logs)を使用してログを取得できます。
