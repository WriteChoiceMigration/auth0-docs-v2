Security Assertion Markup Language（SAML）プロトコルは、2つのエンティティ間でパスワードを使用せずに認証および認可を行うための、オープンスタンダードでXMLベースのフレームワークです。

* **サービスプロバイダー**（SP）は、IDプロバイダーを信頼することに同意し、ユーザーを認証します。
* **IDプロバイダー**（IdP）はユーザーを認証し、サービスプロバイダーに対して、ユーザーが認証されたことを示す認証アサーションを提供します。

Auth0はSAMLプロトコルをサポートしており、IdP、SP、またはその両方として機能できます。これには以下が含まれます。

* SAML2 Webアプリケーション
* SAML SSO統合
* IdPを起点とするSSO
* Microsoft Active Directoryフェデレーションサービス（ADFS）
* SAML要求の署名と暗号化

## サポートされているSAMLバインディングおよびオプション

Auth0は次のSAMLバインディングをサポートします。

* HTTPリダイレクト
* HTTP POST

Auth0は次のSAMLオプションをサポートします。

* WebブラウザーSSOプロファイル
* シングルログアウトプロファイル
* 名前ID管理プロファイル
* 名前IDマッピングプロファイル
