## Display（表示）

[［Tenant Settings（テナント設定）］](%24%7Bmanage_url%7D/#/tenant/)で関連するフィールドを構成済みの場合、汎用エラーページに以下の情報が表示されます。



## パラメーター

認可サーバーは、パラメーターをクエリ文字列としてURLに追加して返します。



このパラメーターはエラーの種類によって異なり、要求固有のものになります。たとえば、要求に`client_id`が含まれておらずエラーが発生した場合、エラーページでクライアントIDは返されません。
