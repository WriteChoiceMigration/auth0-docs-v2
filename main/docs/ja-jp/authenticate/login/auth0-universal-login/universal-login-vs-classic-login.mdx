[Auth0のユニバーサルログイン](/login/universal-login)には、認可サーバーの重要な機能であるログインフローが備わっています。ユーザーがアプリケーションにアクセスするために本人確認を行う必要がある場合は、ユニバーサルログインにリダイレクトし、Auth0に認証プロセスを処理してもらうことができます。

ユニバーサルログインは、Auth0でホストされる主要なログインソリューションです。[ユニバーサルログイン](/login/universal-login/universal-experience)は、使いやすいカスタマイズツールを備えており、よりシンプルで高速なエクスペリエンスをエンドユーザーに提供します。また、Auth0では[クラシックログイン](/login/universal-login/classic-experience)もサポートしており、このホストされたログインエクスペリエンスではカスタマイズにJavaScriptコントロールを使用します。
