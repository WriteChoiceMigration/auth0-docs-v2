Auth0は、以下のエンタープライズプロバイダーに対応しており、すぐに使用することができます。

* [Active Directory/LDAP](/active-directory-ldap)
* [ADFS](/adfs)
* [Azure Active Directory Native](/azure-active-directory-native)
* [Google Workspace](/google-apps)
* [OpenID Connect](/oidc)
* [Okta](/okta)
* [PingFederate](/ping-federate)
* [SAML](/saml)
* [Azure Active Directory](/azure-active-directory/v2)



パートナーが対応しているエンタープライズ接続を[Auth0 Marketplace](https://marketplace.auth0.com/features/enterprise-connections)で探すこともできます。
