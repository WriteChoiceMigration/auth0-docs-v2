[AD/LDAP接続](/extensions/ad-ldap-connector)の資格情報キャッシュは、Auth0 Dashboardを使用して接続レベルで無効にすることができます。

1. [［Auth0 Dashboard］>［Authentication（認証）］>［Enterprise（エンタープライズ）］](%24%7Bmanage_url%7D/#/connections/enterprise)に移動して、**［Active Directory/LDAP］**の接続タイプを選択します。
2. 編集したい接続の名前を選択します。
3. **［Disable Cache（キャッシュを無効にする）］**を有効にします。
