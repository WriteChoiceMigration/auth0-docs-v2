Auth0は、識別子（メール、ユーザー名、または電話番号）とパスワードを使ってユーザーを認証するためのデータベース接続を提供します。これらの資格情報はAuth0ユーザーストアまたは独自のデータベースに安全に保存されます。

新しいデータベース接続の作成や、既存のデータベース接続の管理は、[［Auth0 Dashboard］ > ［Authentication（認証）］ > ［Database（データベース）］](%24%7Bmanage_url%7D/#/connections/database)で行えます。
