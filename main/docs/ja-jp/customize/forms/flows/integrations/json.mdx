以下に記載のJSONアクションは、JSONの作成、解析、文字列変換を行えるようにします。

## JSONオブジェクトを作成する

JSONオブジェクトを作成します。これは、フローにある他のアクションでコンテンツを再利用するのに便利です。

<Frame>![](/images/cdy7uua7fh8z/1zJEENRPI2uONuCApPY98p/f1fce88a42fb13fb9856c95c562bd506/createjsonobject.png)</Frame>

### 入力設定

<table class="table">
<thead>
<tr>
<th><b>Parameter</b></th>
<th><b>Description</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>Body</td>
<td>
              The JSON object.
          </td>
</tr>
</tbody>
</table>

### 出力オブジェクト

<table class="table">
<thead>
<tr>
<th><b>Property</b></th>
<th><b>Type</b></th>
<th><b>Description</b></th>
</tr>
</thead>
<tbody>
<tr>
<td><code>result</code></td>
<td>Object</td>
<td>The JSON object.</td>
</tr>
</tbody>
</table>

### 出力オブジェクトの例

```json lines
{
	"result": {
		"name": "John Doe"
	}
}
```
