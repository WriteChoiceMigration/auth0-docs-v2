このXMLアクションのリストを使用すると、JSONオブジェクトをXML文字列に変換したり、XML文字列をJSONオブジェクトに変換したりできます。

## JSONをXMLに変換する

JSONオブジェクトをXML文字列に変換します。

<Frame>![](/images/cdy7uua7fh8z/3GW2EhBMIX3YDPaqc3rXrZ/7706a863700df51376d645a7d9292a27/convertjsontoxml.png)</Frame>

### 入力設定

<table class="table">
<thead>
<tr>
<th><b>Parameter</b></th>
<th><b>Description</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>JSON (required)</td>
<td>
              The JSON object that will be converted to an XML string.
          </td>
</tr>
</tbody>
</table>

### 出力オブジェクト

<table class="table">
<thead>
<tr>
<th><b>Property</b></th>
<th><b>Type</b></th>
<th><b>Description</b></th>
</tr>
</thead>
<tbody>
<tr>
<td><code>xml</code></td>
<td>String</td>
<td>The converted XML string.</td>
</tr>
</tbody>
</table>

### 出力オブジェクトの例

```json lines
{
  "xml": "<?xml version=\"1.0\" encoding=\"UTF-8\" standalone=\"yes\"?>\n<name>John Doe</name>"
}
```






## XMLをJSONに変換する

XML文字列をJSONオブジェクトに変換します。

<Frame>![](/images/cdy7uua7fh8z/42fO7BelEN4bnlUj70WhV8/e53ada552f7cc05c192466db4777471a/covertxmltojson.png)</Frame>

### 入力設定

<table class="table">
<thead>
<tr>
<th><b>Parameter</b></th>
<th><b>Description</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>XML (required)</td>
<td>
              The XML string that will be converted to a JSON object.
          </td>
</tr>
</tbody>
</table>

### 出力オブジェクト

<table class="table">
<thead>
<tr>
<th><b>Property</b></th>
<th><b>Type</b></th>
<th><b>Description</b></th>
</tr>
</thead>
<tbody>
<tr>
<td><code>object</code></td>
<td>Object</td>
<td>The converted JSON object.</td>
</tr>
</tbody>
</table>

### 出力オブジェクトの例

```json lines
{
  "object": {
    "name": "John Doe"
  }
}
```
