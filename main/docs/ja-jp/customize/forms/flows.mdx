フローを使用すると、フォーム内で実行可能なサーバー側のロジックを視覚的に構築し、収集した情報の処理、APIによるサービスの統合、自動化のトリガーを行えます。

<Frame>![Dashboard > Forms > Flows](/images/cdy7uua7fh8z/6u6Huh5CbH3GEQJv3AH7cx/e49ea613b2c46a43c83c89ffecddbafe/Flows.png)</Frame>

フォームでフローアクションを使用して次のことが可能です。

* フォームの手順に関するカスタムのサーバー側検証を作成する。
* カスタムHTTP要求を作成して、独自のAPIと統合する。
* すぐに導入できる統合を使用して、サードパーティと統合する。
* ユーザーの入力またはプロパティに基づいて異なるビジネスロジックを実行する。
* Validate data using <Tooltip tip="JSON Web Token (JWT): Standard ID Token format (and often Access Token format) used to represent claims securely between two parties." cta="View Glossary" href="/docs/glossary?term=JSON+Web+Tokens">JSON Web Tokens</Tooltip>.
