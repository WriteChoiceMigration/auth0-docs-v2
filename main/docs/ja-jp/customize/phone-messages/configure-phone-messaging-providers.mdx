<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

Phone messaging providers are currently in Early Access. To learn more about Auth0 releases, review [Product Release Stages](/docs/troubleshoot/product-lifecycle/product-release-stages).

</Callout>

Auth0では、SMSおよび音声通信を管理、監視、トラブルシューティングするために独自の電話メッセージプロバイダーを設定することが可能です。テナントごとに、すべてのSMSおよび音声通信のために1つの電話プロバイダーのみを設定できます。

Auth0は現在、以下の電話メッセージプロバイダーをサポートしています。

* [Twilio](/docs/customize/phone-messages/configure-phone-messaging-providers/configure-twilio-as-a-phone-messaging-provider)
* [Custom phone provider](/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider)

You can also configure a custom phone provider using Actions. To learn more, read [Configure a Custom Phone Provider](/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider) and [Actions Triggers: custom-phone-provider Event Object](/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider/actions-triggers-custom-phone-provider-event-object).
