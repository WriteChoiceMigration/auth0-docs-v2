独自のSMTPメールプロバイダーを設定して、メール通信の管理、監視、トラブルシューティングを行い、メールテンプレートをカスタマイズすることができます。



## カスタムSMTPサーバーを構成する

1. [［Auth0 Dashboard］>［Branding（ブランディング）］>［Email Provider（メールプロバイダー）］](%24%7Bmanage_url%7D/#/templates/provider)に移動します。
2. **［Use my own email provider（独自のメールプロバイダーを使用する）］**を有効にします。
3. **［SMTP］**を選択します。
4. **［From（送信元）］**メールアドレスを入力し、SMTP サーバーの**［Host（ホスト）］**、**［Port（ポート）］**、**［Username（ユーザー名）］**、**［Password（パスワード）］**を入力します。一般的なポートには25と587があります。多くのプロバイダーがこのポートに制限を設けているため、可能であればポート25の使用は避けてください。
5. **［Save（保存）］**をクリックします。

## メール配信のテスト

テストメールを送信するには、**［Send Test Email（テストメールの送信）］**ボタンを使用します。すべてが正しく構成されていると、確認メールを受け取ります。数分経ってもメールを受信しない場合は、[Auth0ログ](%24%7Bmanage_url%7D/#/logs)でエラーを確認してください。

SMTPを使用すると、ユーザーにスパムメールを送信することなく、セットアップが正しく動作しているかテストできるテストサービスを簡単にセットアップできます。詳細については、[［Set Up Test SMTP Servers（テストSMTPサーバーのセットアップ）］](/email/testing)を参照してください。
