ダッシュボードでは、（ルール、アクション、カスタムデータベーススクリプト、カスタムのソーシャル接続に加えて）フックのログ情報をリアルタイムで確認できます。

1. [［Auth0 Dashboard］>［Auth Pipeline（Authパイプライン）］>［Hooks（フック）］](%24%7Bmanage_url%7D/)に移動し、フックの横にある鉛筆アイコンを選択します。
2. Hooks Editorの右上にあるログのアイコンを選択します。
