DashboardやManagement APIを使用して、任意の[［Hook（フック）］](/hooks)に対して複数のフックシークレットを作成できます。

フックシークレットは、[Deploy Command-Line Interface (CLI) ツール](/deploy/deploy-cli-tool)を使用してインポートやエクスポートすることができます。



## Dashboardを使用してフックシークレットを表示します

1. [［Auth0 Dashboard］](%24%7Bmanage_url%7D/)の[［Hooks（フック）］](%24%7Bmanage_url%7D/#/hooks)ページに移動し、編集するフックの横にある鉛筆アイコンをクリックします。
2. フックエディターで、レンチアイコンをクリックし、**［Secrets（シークレット）］**をクリックします。
3. **［Add Secret（シークレットを追加）］**をクリックします。
4. シークレットのわかりやすい名前と値を入力し、**［Save（保存）］**をクリックします。

## Management APIを使用してフックシークレットを作成する

[Add Hook Secrets エンドポイント](/api/management/v2/#!/Hooks/post_secrets)に` POST`呼び出しを行います。`MGMT_API_ACCESS_TOKEN`、`HOOK_ID`、`HOOK_SECRET_KEY`、および`HOOK_SECRET_VALUE`のプレースホルダー値は、それぞれManagement APIアクセストークン、フックID、およびHook key-valueペアに置き換えてください。
