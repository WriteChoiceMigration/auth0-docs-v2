Hooksは、シークレットを安全に保管すると同時に、コードで便利に使えるようにする統合シークレット管理の機能を果たします。



## Hooks Secretsを管理する

Hook Secretsは、ダッシュボードまたはManagement APIから作成・更新・削除・表示できます。詳細については、以下をお読みください。

* [Hook Secretsを作成する](/hooks/hook-secrets/create-hook-secrets)
* [Hook Secretsを更新する](/hooks/hook-secrets/update-hook-secrets)
* [Hook Secretsを削除する](/hooks/hook-secrets/delete-hook-secrets)
* [Hook Secretsを表示する](/hooks/hook-secrets/view-hook-secrets)
