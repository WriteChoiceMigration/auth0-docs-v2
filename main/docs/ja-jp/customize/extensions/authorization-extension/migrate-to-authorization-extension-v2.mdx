認可拡張機能のバージョン1と2にある主だった変更は、**［Applications（アプリケーション）］**セクションが削除されたことです。このセクションは、アクセス制御のポリシー定義で使用されていた場合など、内在的な複雑性を簡素化させるために削除されました。そのようなユースケースには、[ルール](/rules)の使用が推奨されます。

## 拡張機能のバージョンをアップグレードする

認可拡張機能をアップグレードするには、[［Auth0 Dashboard］>［Extensions（拡張機能）］](%24%7Bmanage_url%7D/#/extensions)に移動して、**［Installed Extensions（インストール済みの拡張機能）］**ビューを選択します。

認可拡張機能の行に、最新バージョンへのアップグレードを開始するリンクがあります。
