シングルサインオン（SSO）ダッシュボードは、多くのユーザーになじみのある問題を解決するためにデザインされたウェブアプリケーションです。あらゆる規模の組織は、会計、HR、開発、サポートなど様々なビジネス機能に対処するために多種多様なアプリケーションを維持しています。それぞれのアプリケーションのユーザー名とパスワード、そしてログインURLをすべて覚えておくことは非常に困難です。この拡張機能を使用すれば、複数のエンタープライズアプリケーションでユーザーのSSOログインを有効にして、認証エクスペリエンスを簡略化できます。それにより、SSOログインでユーザーを認証できるクラウドサービスの全リストを作成できます。



SSO ダッシュボードは、2種類のユーザーをサポートします。

* **［Users（ユーザー）］**は、ダッシュボードにログインして、SSOでサインインするアプリケーションを選択できます。
* **［Admins（管理者）］**は、ユーザーに表示されるアプリケーションを構成できます。

この拡張機能を構成する手順は次のとおりです。

1. [SSOダッシュボードアプリケーションを作成する](/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)
2. [SSOダッシュボード拡張機能をインストールする](/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)
3. [SSO ダッシュボードにアプリケーションを追加する](/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)
4. [SSO ダッシュボードのアプリケーションを更新する](/extensions/single-sign-on-dashboard-extension/update-applications-on-the-sso-dashboard)

また、[GitHubでこのSSO ダッシュボード拡張機能を表示する](https://github.com/auth0-extensions/auth0-sso-dashboard-extension)こともできます。
