Auth0拡張機能を使うと、アプリケーションのインストールやコマンド・スクリプトの実行を行ってAuth0の基本製品の機能性を拡張することができます。各拡張機能は互いに独立しており、Auth0の[レート制限ポリシー](/support/policies/rate-limit-policy)の対象となります。Auth0では、テナントごとに拡張機能を定義しているため、データは`tenant\extension`のペアで保存されます。
