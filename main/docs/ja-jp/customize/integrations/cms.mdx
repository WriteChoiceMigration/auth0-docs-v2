Auth0はコンテンツ管理システム（CMS）のプラグインや拡張機能を提供し、使用しているCMSのセットアップをAuth0アカウントに統合できるようにしています。これらのプラグインは、エンタープライズのシングルサインオン（SSO）、ソーシャルログイン、ユーザーとパスワードのログインをすべてのインスタンスとプラットフォームで利用できるようにします。

ログイン機能は、新しいログインウィジェット（Auth0機能搭載）を通して実装され、以下が利用できるようになります。

* エンタープライズディレクトリ（LDAP、AD、Google Workspace、Office365、SAMLプロバイダー）でのSSO
* 複数のWordPressアカウントでユーザーとパスワードを共有したSSO
* 30以上の[ソーシャルプロバイダー](/identityproviders)
* User Management（ユーザー管理）Dashboard
* 任意の二要素認証
* WordPressと他のアプリケーション間のSSO
* レポートと分析
