Office 365は廃止されました。Offce365 ConnectionsをWindows Azure AD Connectionsに移行する必要があります。

最近から、弊社はOffice365を使用してユーザーをサポートしています。Office365は常にWindows Azure ADをバックエンドで使用していましたが、Windows Azure ADで「アプリケーション」を作成するための良いUIがありませんでした。そのためSeller Dashboardで作成しなければなりませんでした。今後、MicrosoftはWindows Azure ADを使用することを推奨しており、Office365アカウントに関連付けられたディレクトリやアプリケーションを簡単に作成できるようになっています。

## Azureへの移行方法

1. Create a Windows Azure AD subscription (free)
2. Create a Directory (that will be associated with your Office365 account) and an application. To learn more, read [Connect Your App to Microsoft Azure Active Directory](/docs/authenticate/identity-providers/enterprise-identity-providers/azure-active-directory/v2).

アプリケーションで `user_id`を使用していた場合、`office365|....some-guid....`は`waad|...email....`に変更されることにご注意ください。
