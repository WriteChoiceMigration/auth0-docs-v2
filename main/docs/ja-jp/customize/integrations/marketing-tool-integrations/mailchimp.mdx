この記事では、Auth0からユーザーデータをCSVファイルとしてエクスポートし、[MailChimp dashboard](https://login.mailchimp.com/)にインポートする方法を説明します。

## ユーザーデータファイルを作成する

［Dashboard］の[［Extensions（拡張機能）］](%24%7Bmanage_url%7D/#/extensions)セクションに移動し、**［User Import / Export Extension（ユーザーのインポート/エクスポート拡張機能）］**を開きます。拡張機能ページのメニューから**［Export（エクスポート）］**を選択します。

次に、**［Export Format（エクスポート形式）］**を必要なファイル形式に設定します。MailChimpはCSV形式のファイルインポートを受け付けるため、`［Tab Separated Value file（タブ区切り値ファイル）（*.csv）］`オプションを選択してください。

詳しくは、「[Mailchimpナレッジベース：インポートファイルのフォーマットガイドライン](https://kb.mailchimp.com/lists/growth/format-guidelines-for-your-import-file)」を参照してください。

**［Fields（フィールド）］**セクションの一番上で、それぞれのユーザー属性に対して**［User Field（ユーザーフィールド）］**と**［Column Name（コラム名）］**を提供し、エクスポートに含めてください。MailChimpでは、コラム名`Email Address`がメールフィールドに必要であるため、含めるようにしてください。例：



ユーザーフィールドの追加後、**［Export Users（ユーザーのエクスポート）］**ボタンをクリックし、エクスポートを開始します。エクスポートが完了したら、CSVファイルをダウンロードし、以下のセクションで使用します。

## ユーザーデータファイルのインポート

開始する前に、「[MailChimpナレッジベース：定期購読者をリストにインポートする](https://kb.mailchimp.com/lists/growth/import-subscribers-to-a-list)」にアクセスすると、詳細を確認できます。

MailChimpアカウントにログインし、**Lists（リスト）**ページに移動します。Auth0ユーザーをインポートするリストを選択（[または新しいリストを作成](https://kb.mailchimp.com/lists/growth/create-a-new-list)）します。

1. MailChimpのリストページで、**［Add Contacts（連絡先の追加）］**メニューから**［Import Contacts（連絡先のインポート）］**をクリックします。
2. 連絡先のインポート元となる、`［CSV or tab-delimited text file（CSVまたはタブ区切りテキストファイル）］`オプションを選択します。
3. 前のセクションで、Auth0からエクスポートしたユーザーデータのCSVファイルをアップロードします。MailChimpは、次のページでユーザーデータを解釈します。
4. コラム名とフィールドタイプが正しいことを確認してください。
5. **［Next（次へ）］**ボタンをクリックします。
6. 選択内容を確認し、インポートカテゴリを設定します。
7. **［Import（インポート）］**ボタンをクリックしてユーザーインポートを開始します。

これで作業完了です。MailchimpへのAuth0ユーザーのインポートに成功しました。
