[Auth0 Management API](https://auth0.com/docs/api/management/v2)には、MFA認証方法の管理に使用できるエンドポイントがいくつかあります。

この方法では、機密アプリケーションを使った認証が用いられます。機密アプリケーションとパブリックアプリケーションの詳細については、「[機密アプリケーションとパブリックアプリケーション](/applications/confidential-and-public-applications)」をお読みください。
