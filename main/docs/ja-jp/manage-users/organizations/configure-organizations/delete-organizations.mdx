[組織](/organizations/organizations-overview)を削除すると、すべてのユーザーのメンバーシップと削除された組織との関連付けが削除されますが、ユーザー自体はテナントから削除されることはありません。



組織の削除はAuth0 DashboardまたはManagement APIを使って行うことができます。

## Auth0 Dashboard

Auth0 Dashboardを使用して組織を削除するには、以下を行います。

1. [［Auth0 Dashboard］>［Organization］](%24%7Bmanage_url%7D/#/organizations)に移動して、削除する組織を選択します。
2. スクロールして**［Danger Zone（危険ゾーン）］**セクションを見つけ、**［Delete（削除）］**を選択し、確定します。

## Mangement API

`DELETE`呼び出しを`Delete Organization`（組織削除）エンドポイントに対して行います。`ORG_ID`と`MGMT_API_ACCESS_TOKEN`のプレースホルダーの値を、それぞれ組織IDとManagement APIのアクセストークンに置き換えてください。
