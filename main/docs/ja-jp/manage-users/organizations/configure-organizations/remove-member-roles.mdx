Auth0 DashboardまたはManagement APIを使用すると、[組織](/organizations/organizations-overview)メンバーからロールを削除することができます。

## Auth0 Dashboard

Auth0 Dashboardを使って組織メンバーからロールを削除するには、以下を行います。

1. [［Auth0 Dashboard］>［Organization］](%24%7Bmanage_url%7D/#/organizations)に移動して、メンバーシップを構成したい組織を選択します。
2. **［Members（メンバー）］**ビューを選択し、ロールを削除したいメンバーの名前をクリックします。
3. 削除したいロールを見つけて、ゴミ箱アイコンをクリックし、確認します。

## Management API

Management APIを使って組織メンバーからロールを削除するには、次を行います：
 
`DELETE`呼び出しを`Delete Organization Member Roles`（組織メンバーロール削除）エンドポイントに対して行います。`ORG_ID`、`MGMT_API_ACCESS_TOKEN`、`USER_ID`、`ROLE_ID`のプレースホルダーをそれぞれ組織ID、管理APIアクセストークン、ユーザID、ロールIDで置き換えます。
