本ガイドでは、[Role-based Access Control（RBAC）](/authorization/concepts/rbac)システムでアクセス許可を管理する方法をご紹介します。これらのアクセス許可は、API Authorization Core機能セットで使用されます。

アクセス許可を管理するためのさまざまな機能が提供されており、Auth0 Dashboardまたは Auth0 Management APIを通じて利用できます。

* [APIのアクセス許可を追加する](/apis/add-api-permissions)
* [APIのアクセス許可を削除する](/apis/delete-api-permissions)
* [アクセス許可をロールに追加する](/rbac/roles/add-permissions-to-roles)
* [ユーザーに権限を割り当てる](/rbac/rbac-users/assign-permissions-to-users)
* [ロール権限を表示する](/rbac/roles/view-role-permissions)
* [ユーザーのアクセス許可を表示する](/rbac/rbac-users/view-user-permissions)
* [ロール権限を削除する](/rbac/roles/remove-permissions-from-roles)
* [ユーザーのアクセス許可を削除する](/rbac/rbac-users/remove-permissions-from-users)
