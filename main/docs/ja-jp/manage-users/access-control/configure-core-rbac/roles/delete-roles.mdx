Auth0 DashboardまたはManagement APIを使用して、[ロール](/rbac)を削除できます。ロールはAPIの認可コア機能セットで使用されます。



## Dashboard

1. [［Dashboard］>［User Management（ユーザー管理）］>［Roles（ロール）］](%24%7Bmanage_url%7D/#/roles)に移動し、ロール名をクリックして表示します。
2. **［Remove this Role（このロールを削除）］** をクリックして確認します。

## Mangement API

`DELETE`呼び出しを[ロール削除エンドポイント](https://auth0.com/docs/api/management/v2#!/Roles/delete_roles_by_id)に対して行います。必ず、`ROLE_ID`と`MGMT_API_ACCESS_TOKEN`のプレースホルダーの値を、それぞれご自身のロールIDとManagement APIのアクセストークンに置き換えてください。
