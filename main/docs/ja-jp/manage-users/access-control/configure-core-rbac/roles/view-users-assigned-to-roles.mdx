Auth0 DashboardまたはManagement APIを使用して、[ロール](/authorization/concepts/rbac)に割り当てられたユーザーを表示することができます。ロールはAPIの認可コア機能セットで使用されます。

## 前提条件



## Dashboard

1. [［Dashboard］>［User Management（ユーザー管理）］>［Roles（ロール）］](%24%7Bmanage_url%7D/#/users)に移動し、ロール名をクリックして表示します。
2. **［Users（ユーザー）］**タブをクリックします。

それぞれのユーザーに以下の情報が表示されます。



## Mangement API

ロールに割り当てられたユーザーのリストを取得するには、「[ロールのユーザーを取得する](https://auth0.com/docs/api/management/v2#!/Roles/get_role_user)」を参照してください。
