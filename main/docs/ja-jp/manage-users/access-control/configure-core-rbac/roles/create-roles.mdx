Auth0のDashboardまたはManagement APIを使用して、[ロール](/rbac)を作成することができます。ロールは、API Authorization Core機能セットで使用できます。

## 前提条件

* 
* Auth0 Dashboardで[APIをセットアップ](/get-started/set-up-apis)します。
* 権限は事前に定義された値から選択されます。権限のリストが空白の場合は、APIに[権限を追加する](/apis/add-api-permissions)必要があります。

## Dashboard

1. [［Dashboard］>［User Management（ユーザー管理）］>［Roles（ロール）］](%24%7Bmanage_url%7D/#/roles)に移動して、**［Create Role（ロールの作成）］**をクリックします。
2. ロールに名前を付け、説明を追加してから、**［Create（作成）］**をクリックします。

## Mangement API

[ロール作成エンドポイント](https://auth0.com/docs/api/management/v2#!/Roles/post_roles)に`POST`呼び出しを行います。`MGMT_API_ACCESS_TOKEN`、`ROLE_NAME`、`ROLE_DESC`のプレースホルダーをそれぞれManagement APIのアクセストークン、ロール名、ロールの説明に置き換えます。
