Auth0の[Management API](/api/management/v2)を使用したユーザー検索を使うと、ユーザープロファイルの詳細を取得できます。検索結果は、[表示](/users/user-search/view-search-results-by-page)、[保存](/users/user-search/sort-search-results)、および[エクスポート](/users/import-and-export-users/bulk-user-exports)することができます。

ほとんどのユーザープロファイルフィールドは、[IDトークン](/security/tokens/id-tokens)の一部として返されることも、Authentication APIの[`/userinfo`エンドポイント](/api/authentication#get-user-info)からの応答に含まれることもありません。

Auth0でユーザーを検索する際には、IDやメール、その他の条件で複数のエンドポイントを検索することができます。

## 検索結果の用語

ユーザー検索結果の説明では以下の用語を使用します。

* **結果整合性**：検索結果に最近完了した書き込み操作が反映されていない可能性があります。ただし、しばらく待ってから要求し直すと、応答で更新されたデータが返されます。
* **即時整合性**：検索結果には、要求の直前に発生したものも含め、成功したすべての書き込み操作の結果が反映されます。
