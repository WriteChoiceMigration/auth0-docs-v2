ユーザーに割り当てられたデバイスは、Auth0 Dashboardを使用してリンクを解除できます。これにより、選択したデバイスのリフレッシュトークンが取り消されます。

1. [［Dashboard］>［User Management（ユーザー管理）］>［Users（ユーザー）］](%24%7Bmanage_url%7D/#/users)に移動し、ユーザー名をクリックします。
2. **［Devices（デバイス）］**ビューをクリックしてから、リンクを解除したいデバイスの隣にあるゴミ箱アイコンをクリックして、確認します。
