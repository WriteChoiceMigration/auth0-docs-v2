1. [［Dashboard］>［User Management（ユーザー管理）］ - ［Users（ユーザー）］](%24%7Bmanage_url%7D/#/users)に移動します。
2. 削除するユーザーの名前をクリックします。
3. **［Details（詳細）］**タブで一番下までスクロールして、**［Delete user（ユーザー削除）］**の横にある**［Delete（削除）］**をクリックします。

ポップアップ画面に、アクションを元に戻すことはできないという警告と、ユーザーを削除するかどうかを確認するメッセージが表示されます。**［Yes, Delete It（はい、削除します）］**ボタンをクリックして確認します。これでユーザーが完全に削除されます。
